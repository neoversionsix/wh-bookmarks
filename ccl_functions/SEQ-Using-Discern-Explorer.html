The SEQ( ) function is used to define a unique identifier when inserting new records into a table. The function returns the current or the next value in the sequence that is specified. Unlike other functions, SEQ( ) can only be used in the following three ways:

  1. Select list in a SELECT command
  2. Set clause of an insert
  3. Set clause of an update

The SEQ( ) function must be used with an Oracle table. It cannot be used with an ISAM (non-Oracle) table. For example, use DUAL not DUMMYT.

Syntax

SEQ(seq_name, val)

Metasymbols

seq_name
A sequence name for a specified table.

val
CURRVAL or NEXTVAL are used to determine which sequence number to return.

Examples

The following example uses SEQ() to set the ID field when inserting records into a custom RDBMS table:

INSERT INTO CUST_TABLE C
	(C.ID, C.DT, C.NAME)
	(SELECT
	  SEQ(PERSON_CUSTOM_SEQ,NEXTVAL),;SEQUENCE NUMBER
	  CNVTDATETIME(CURDATE,CURTIME3),
	  CNVTUPPER(P.NAME_LAST)
    	FROM PERSON P
    	WHERE P.UPDT_DT_TM > CNVTDATETIME(CURDATE -2,0)
    	WITH NOCOUNTER)

The following example uses a Select from Dual to set a global variable equal to the value returned from the SEQ() function:

declare gv_next_id = f8

select into "nl:"
	se_id = seq(person_cust_seq,nextval)
from dual
detail
	gv_next_id = se_id
with nocounter

call echo(gv_next_id)
