The ASSIGN( ) function moves the assign_from_expr to the assign_to_expr. The VALIDATE function is allowed around the assign_to_expr.

Syntax

ASSIGN(assign_to_expr, assign_from_expr)


Metasymbols

assign_to_expr
An expression.

assign_from_expr
An expression.

Examples

In the following example, stat is set to logic(curindex) on return, which is the result of the sub call.

SET STAT = ASSIGN(LOGIC[CURINDEX],SUB(P1,P2))


In the example below, using the ASSIGN( ) function allows you to apply the CNVTALPHANUM( ) function to an RDB column in the WHERE clause. The DUMMYT table (D2) is used to assign the value of CNVTALPHANUM(CGES.EVENT_ST_NAME) to VAR1. Then, VAR1 is used to join to SC.EVENT_SET_NAME_KEY. PLAN CGES
JOIN D2 WHERE ASSIGN(VAR1,CNVTALPHANUM(CGES.EVENT_SET_NAME) ) != " "
JOIN SC	WHERE VAR1 = SC.EVENT_SET_NAME_KEY
