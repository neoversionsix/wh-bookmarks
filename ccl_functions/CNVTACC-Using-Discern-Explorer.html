The CNVTACC( ) function returns a formatted accession number as a C25 data type. The accession number format is specified using DB Accession Setup (AccessionSetup.exe).

Syntax

CNVTACC(<raw_accession_string>)

Metasymbols

raw_accession_string
An unformatted accession number string.

Examples

In the example below, the CNVTACC() function is used to format the ACCESSION field of the ACCESSION table. If A.ACCESSION is 00000MA19980000161, then 00000-MA-1998-0000161, MA-1998-0000161, or another format is echoed based on the format specified in DB Accession Setup.

Call Echo(CNVTACC(A.ACCESSION))
