Returns the four-character phonetic translation of the textual expression.

Syntax

SOUND(text_expr[,length])

Metasymbol

text_expr

A text expression.

length

An integer representing length of returned string. The default is 4. The values 1 through 4 are valid.

Examples

In this example, a four character sound string is returned with a printed result of S530.

CALL PRINT(sound("SMITH",4))

In these examples, the printed result is F260.

CALL PRINT(SOUND("FISHER"))
CALL PRINT(SOUND("FICHER"))
