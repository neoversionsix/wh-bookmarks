Note

This page contains instructions for using the EVALUATE function. See EVALUATE Command Using Discern Explorer Debugger for information on using the EVALUATE command.

The EVALUATE( ) function allows you to return an expression based on a condition. Similar functionality is used with theÂ CASE or IF commands.

Syntax

EVALUATE(case_expr,case_value,return_value {,case_value,return_value} [,return_value])

Metasymbols

case_expr

An expression that is being compared to the case_value.

case_value
A value that is compared to the case_expr. If the condition is true, the return_value is returned.

return_value
The value that is returned if the case_expr is equal to the case_value. The final return_value is used as a default if none of the conditions are true.

Examples

The following example demonstrates that if the value in the P.PHYSICIAN_IND column is 1, PHYS_IND is set to "YES". If the value is 0, PHYS_IND is set to "NO".

PHYS_IND = EVALUATE(P.PHYSICIAN_IND,1,"YES",0,"NO")

The following example demonstrates that GENDER is equal to "UNKNOWN" if P.SEX_CD is not equal to MALE_VAR or FEMALE_VAR.

GENDER = EVALUATE(P.SEX_CD,MALE_VAR,"MALE",FEMALE_VAR,"FEMALE","UNKNOWN")
