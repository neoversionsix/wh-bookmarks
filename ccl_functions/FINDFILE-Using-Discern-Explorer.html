FINDFILE( )
Determines if a specific file exists and if you have access to the file. Returns a 1 if the file is found and you have access to the file. Returns a 0 if the file is not found or you do not have read access to the file. 

Syntax

FINDFILE(file_name[,mode,[casemode]])

Metasymbols

file_name 

The file name to be searched for. By default, Discern Explorer looks for the file in CCLUSERDIR, unless a path for the file location is included in the file_name.  By default, the file_name is implicitly converted to lowercase on Unix file systems and uppercase on AXP file systems.  The casemode  option can be used to override the implicit case conversion. Beginning in Discern Explorer versions 8.17.5 and 9.05.5, spaces are allowed in the file_name.

mode

Description

0

Check if the file exists (default).

1

Check if the file exists and execute access is allowed.

2

Check if the file exists and write access is allowed.

3

Check if the file exists and execute and write access is allowed.

4

Check if the file exists and read access is allowed.

5

Check if the file exists and execute and read access is allowed.

6

Check if the file exists and write and read access is allowed.

7

Check if the file exists and execute, write, and read access is allowed.

Release Consideration

casemode is available in Discern Explorer version 8.7.2 and higher.

casemode

Can be used to override the implicit case conversion of the file_name.

Mode

Description

0

Find lowercase file on Unix domains or uppercase file on AXP domains (default).

1

Find file using exact match of file name on Unix domains else uppercase file on AXP domains.

Examples

If the file test.prg is found in CCLUSERDIR, stat is set to 1.

SET STAT = FINDFILE("TEST.PRG") GO

If the file test_report.prg is found in cust_script, and you have read access to test_report.prg, stat is set to 1.

SET STAT = FINDFILE("CUST_SCRIPT:TEST_REPORT.PRG",4) GO

The following example uses mode 1 for the casemode parameter to perform a case sensitive search for the the Test.prg file.

SET STAT = FINDFILE("Test.prt",0,1) GO
