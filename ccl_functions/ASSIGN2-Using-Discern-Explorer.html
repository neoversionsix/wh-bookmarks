You can use the ASSIGN2( ) function to preassign an expression, then assign the expression to the variable.

Syntax

ASSIGN2(<preassign_expr>,<to_expr>,<from_expr>)

Metasymbols

preassign_expr
An expression.

to_expr
An expression.

from_expr
An expression.

Examples

In the following example increment curindex before setting logic(curindex) to result of sub call.

SET STAT = ASSIGN2(ASSIGN(CURINDEX,CURINDEX+1),LOGIC[CURINDEX],SUB(P1,P2))
