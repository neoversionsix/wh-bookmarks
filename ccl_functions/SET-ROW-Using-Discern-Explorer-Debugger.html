The SET ROW command determines the number of rows of the source to be displayed in the Debugger window. This command allows you to override the default number of rows displayed at each break point.

Syntax

S[ET] R[OW] source_rows

Metasymbol

source_rows
The number of rows of the source to be displayed in the Debugger window. The default is six rows.

Example

The following example sets the number of rows to be displayed in the Debugger window to 12:

S R 12
{"serverDuration": 471, "requestCorrelationId": "ad30b476eb097361"}
