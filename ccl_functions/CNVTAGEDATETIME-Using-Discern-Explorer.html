CNVTAGEDATETIME( ) converts an age value in years, months, weeks, or days to an RDB date and time. Always use zero (0) for any argument not used.

Syntax

CNVTAGEDATETIME(year, month, week, day)

Metasymbols

year

A number representing the age in years.

month

A number representing the age in months.

week

A number representing the age in weeks.

day

A number representing the age in days.

Examples

In the example below, start_dt is set to the RDB date and time for 60 years and end_dt is set to the RDB date and time for 70 years. You can then use these expressions to qualify on patients who are between 60 and 70 years of age.

SET START_DT = CNVTAGEDATETIME(60,0,0,0)
SET END_DT = CNVTAGEDATETIME(70,0,0,0)
WHERE P.BIRTH_DT_TM BETWEEN CNVTDATETIME(END_DT) AND CNVTDATETIME(START_DT)
