The MONTH() function allows you to return the integer month of a Julian date or an rdb_date_time column. An integer from 1 to 12 is returned.

Syntax

MONTH(julian_date)
MONTH(rdb_date_time)

Metasymbols

julian_date
An expression that is anÂ Internal System Date.

rdb_date_time
A column that stores a datetime or function that returns a datetime.

Examples

select
	date = P.ACTIVE_STATUS_DT_TM,
	active_month = month(P.ACTIVE_STATUS_DT_TM),
	cur_month = month(curdate),
	fun_month = month(cnvtdatetime(curdate,curtime3)),
	aug_month = month(cnvtdate(08152023))
from
	person p
with maxrec = 10
