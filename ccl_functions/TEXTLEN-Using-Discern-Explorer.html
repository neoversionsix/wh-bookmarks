Returns the variable length of a VC expression.

Syntax

TEXTLEN(expression)

Metasymbol

expression

A textual expression.

Examples

The following examples incorporate the TEXTLEN( ) function:

DATA1 = TEXTLEN(TEXT_1)
DATA2 = TEXTLEN(TRIM(P.NAME_FULL_FORMATTED) )
