LOGICAL( ) returns the translation of an operating system logical (logical_name).

Syntax

LOGICAL(logical_name[,return_length[,mode]])

Metasymbols

logical_name
The name of the logical to be translated.

return_length

An integer value to specify the length of the returned value. If not specified, the default return length is 80 characters. Return_length was introduced in version 8.8.3 of Discern Explorer.

mode

 Mode Description

0

search uppercase then lowercase (default)

1

search case sensitive

2

search uppercase

3

search lowercase

Mode was introduced in version 8.8.3 of Discern Explorer.

Examples

In this example, TESTDIR is set equal to the path for the CCLUSERDIR logical.

TESTDIR = LOGICAL("CCLUSERDIR")
