ISNUMERIC( ) verifies that expressions are numeric. It returns a 1 if the expression is an integer, 2 if it is a floating decimal, or 0 if neither apply. The optional skip_char argument can be used to skip a single character such as a dollar sign.

Syntax

ISNUMERIC(expression[,skip_char,option])

Metasymbols

expression
Any alpha or numeric character.

skip_char
Any character that is to be to be skipped.Â  The characters "-", "+", ".", ",", "E", or "e" cannot be skipped. Discern Explorer 8.12.6 allows multiple skip characters to be used.

option

OptionDescription
0Default scientific notation is allowed, and blanks are valid
1

Scientific notation is not allowed, and blanks are valid

2Scientific notation is allowed, and blanks are invalid
3Scientific notation is not allowed, and blanks are invalid

Options 2 and 3 were added in Discern Explorer 8.12.6.

Examples

The below statement returns 1.

Isnumeric("123")

The below statement returns 2.

Isnumeric("123.4")

The below statement returns 0 because the dollar sign ($) is not numeric.

Isnumeric("$123")

The below statement returns 1 because the dollar sign ($) is skipped.

Isnumeric ("$123","$")

The below statement returns 2.

Isnumeric ("1.2345e+2")

The below statement returns 0 because scientific notation is not allowed for Option 1.

Isnumeric ("1.2345e+2", "a",1)

The below statement returns 0 because blanks are invalid for Option 2.

Isnumeric ("1.2  345e+2", "$",2)

The below statement returns 2 because blanks are valid for Option 1, and the characters "a" and "b" are skipped.

Isnumeric ("1.2  3a4b5", "ba",1)
