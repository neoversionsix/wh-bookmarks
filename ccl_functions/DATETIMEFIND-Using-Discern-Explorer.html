Returns the beginning or ending datetime value of a specified relative range. For example, find the beginning or ending of the day, week, month, quarter, year, or century.

Syntax

DATETIMEFIND(datetime, datemode, range, timemode )


Metasymbols

datetime

The date and time value to base the calculation on.

datemode:

The date unit to find.

Datemode

Description

"D"

Return begin or end of day

"W"

Return week begin or end day

"M"

Return month begin or end day

"Q"

Return quarter begin or end day

"Y"

Return year begin or end day

"C"

Return century begin or end day

range

Finds the beginning or ending day of the date unit.

Range

Description

"B"

Find beginning

"E"

Find ending

timemode

Â Finds the beginning or ending time of the date unit.

Timemode

Description

"B"

Begin time of day

"E"

End time of day

"P"

Preserve time of day

Examples

The following example will return all patients who were registered for the current week:

DROP PROGRAM DATEFIND_EX GO
CREATE PROGRAM DATEFIND_EX
DECLARE BEGIN_DT = F8
DECLARE END_DT = F8
SET BEGIN_DT = DATETIMEFIND(CNVTDATETIME(CURDATE,0),"W","B","B")
SET END_DT = DATETIMEFIND(CNVTDATETIME(CURDATE,0),"W","E","E")
SELECT
P.PERSON_ID,
P.NAME_FULL_FORMATTED,
E.ENCNTR_ID,
E.REG_DT_TM
FROM PERSON P,
ENCOUNTER E
PLAN P
JOIN E WHERE E.REG_DT_TM BETWEEN
CNVTDATETIME(BEGIN_DT) AND CNVTDATETIME(END_DT)
WITH MAXREC = 10
END
GO
