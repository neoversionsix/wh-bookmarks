This function allows you to initialize all the elements of an array to a value. It returns 1 if the values in array_name are initialized to the value defined in init_value; otherwise, 0 is returned. Using INITARRAY() to initialize a numeric data type array, all positions in the array are assigned the init_value. Using INITARRAY() to initialize a fixed length character array, all positions in the array are assigned the int_value padded to the full fixed length of the array.

INITARRAY() can be used to reset a variable but Cerner recommends using a simple assignment instead. Using INITARRAY() to initialize a numeric data type variable assigns the init_value to the variable. Using INITARRAY() to initialize a fixed length character variable, assigns the int_value padded to the full fixed length of the variable. Using INITARRAY() to initialize a variable length character variable, assigns the int_value to all non-blank characters in the variable.

Syntax

INITARRAY(array_name,init_value)

Metasymbols

array_name
The name of an array created using the array_name[occur{,occur}] = data_type|value syntax. Discern Explorer allows you to define one-, two-, and three-dimensional arrays. See DECLARE and SET Variable for information on creating arrays.

init_value
A one-character text, integer, or real value to which an array is initialized.

Examples

The following example uses INITARRAY() to assign AAAAAAAAAA to all three positions of array_c10:

declare istat = i4 with protect
declare array_c10[3] = c10 with protect
set istat = initarray(array_c10,"A")

The following example uses INITARRAY() to assign 123 to all 10 positions of array_i4:

declare istat = i4 with protect
declare array_i4[10] = i4 with protect
set istat = initarray(array_i4,123)

The following example uses INITARRAY() to assign 123.45 to all 5 positions of array_f8:

declare istat = i4 with protect
declare array_f8[5] = f8 with protect
set istat = initarray(array_f8,123.45)

The following example assigns AAA to vc_var:

declare istat = i4 with protect
declare vc_var = vc with protect
set vc_var = "ABC" 
;recommended technique to reset vc_var to AAA
set vc_var = "AAA"
;using initarray() to reset vc_var to AAA
set istat = initarray(vc_var,"A")

The following example does not assign any value to vc_var since vc_var is not assigned any non-blank values prior to the INITARRAY() call:

declare istat = i4 with protect
declare vc_var = vc with protect go
set istat = initarray(vc_var,"A") go
