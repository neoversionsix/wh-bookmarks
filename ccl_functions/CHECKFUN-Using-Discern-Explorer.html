The CHECKFUN ( ) function allows you to determine if a function exists and the function's type.

ReturnDescription
0Not a function.
1Built in Discern Explorer function.
2Aggregate function.
3Screen function.
4Misc function.
5Callprg.
6Function_name begins with UAR_. Only indicates that the string passed for function_name begins with UAR_. Does not confirm that the UAR exists in the shareable CCLUAR library.
7Subroutine.
8SQL function.
9Report function.
10Map function.

Syntax

CHECKFUN(function_name)

Metasymbols

Function_name

The name of the function. It must be in uppercase.

Examples

call echo(CHECKFUN("ALTERLIST")) ;returns 1
call echo(CHECKFUN("COUNT"))     ;returns 2
call echo(CHECKFUN("ECHO"))      ;returns 4
call echo(CHECKFUN("UAR_"))      ;returns 6
