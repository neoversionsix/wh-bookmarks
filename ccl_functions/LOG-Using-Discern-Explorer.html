The LOG( ) function returns the natural logarithm for numeric_expr, which must be a real number or an integer greater than zero.

Syntax

LOG(numeric_expr[,base])

Metasymbols

numeric_expr
An expression that is an integer or a real number.

base
Introduced in Discern Explorer version 8.7.3, base is used to calculate the log of another base. If base is omitted, the natural log of numeric_expr is returned. Log(numeric_expr,base) is equivalent to Log(numeric_expr) / Log(base) in older versions of Discern Explorer that do not allow the base parameter.

Examples

In the following example, if X = 25, then Y = 3.22:

Y = LOG(X)

In the following example, L is set to 2.302585, and L2 is set to 3.321928:

SET L = LOG(10)
SET L2 = LOG(10,2)
