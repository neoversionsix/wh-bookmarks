Returns the phonetic transformation of a textual expression.

Syntax

SOUNDEX(text_expr, [length [,option]])

Metasymbols

text_expr

A text expression.

length

An integer representing the length of the returned string. Valid values are 1 through 8. The default value is 8.

option

Value

Description

1

Use the English version of soundex algorithm for transformation; modified Knuth (default)

2

Use the primary metaphone algorithm for transformation; better for English words

3

Use the secondary metaphone algorithm for transformation; better for non-English words

4

Use the Knuth soundex algorithm which matches Oracle version

5

Use the original Knuth soundex algorithm

Examples

In the following example, the printed result for both uses of the SOUNDEX( ) function is F2600000:

CALL PRINT(SOUNDEX("FISHER"))
CALL PRINT(SOUNDEX("FICHER"))

In the following example, the output of the CALL ECHO( ) for both uses of the SOUNDEX( ) function is F265:

CALL ECHO(SOUNDEX("Fisherman",4))
CALL ECHO(SOUNDEX("Fichermann",4))

In the following example, the output of the CALL ECHO( ) is J5600000:

CALL ECHO(SOUNDEX("Januar",8,1))

In the following example, the output of the CALL ECHO( ) is JNR00000:

CALL ECHO(SOUNDEX("Januar",8,2))

In the following example, the output of the CALL ECHO( ) is ANR00000:

CALL ECHO(SOUNDEX("Januar",8,3))
