The INITREC() function clears a Record structure and deallocates all of the memory held by it. The functionality is the similar to the FREE RECORD command with the exception that the record's definition is retained. The INITREC( ) function is more efficient than the FREE RECORD command, and much more efficient than using the ALTERLIST() function to clear the record when it is reused. The function returns 1 if the record was cleared, 0 if not cleared. Certain server-owned records are not allowed to be cleared.

Syntax

INITREC(record_name)

Metasymbols

record_name
The name of the Record structure for which memory is cleared and deallocated. Discern Explorer version 8.9.0 introduced the ability to use a namespace with the record_name. 


Examples

set stat = initrec(test_rec)
