The BUILD2( ) function allows you to return an expression concatenated with integer and real values treated as strings. The function acts as a concatenate function for combining character expressions with numeric expressions while maintaining leading and trailing spaces for character strings. Float values are converted to a right justified 14 character string rounded to two decimal places. If the converted value exceeds 14 characters, it is truncated to the first 14 characters. Consider using BUILD2CHK( ) instead of BUILD2() for float values. BUILDCHK2() displays asterisks if the converted float value exceeds 14 characters.  Integer values are converted to a right justified 11 character string.

Discern Explorer version 8.12.3 increases the semantic check length from 20 to 24 characters when the BUILD2( ) function is used to pass a parameter with an F8 datatype as an argument to a subroutine call.

In Discern Explorer versions 9.03.0 - 9.03.2 and 8.15.0 - 8.15.2 leading spaces are trimmed from converted float and integer values.

In Discern Explorer versions 9.05.5 and 8.17.5 float values are rounded to two decimal places to match the ROUND() function.

Syntax

BUILD2(expression{,expression})

Metasymbols

Expression
A value of any data type.

Examples

In this example, the result is ABC          12 DEFG       3456 HIJ.

EX=BUILD2("ABC ",12," DEFG",3456," HIJ")

In this example, the result might be Mann, Nicole-06/11/97. The TRIM( ) function is used to trim the trailing spaces from the filed P.NAME_FULL_FORMATTED. The same effect could be created by using the BUILD( ) function.

NAME_BDATE=BUILD2(TRIM(P.NAME_FULL_FORMATTED),"-", FORMAT(P.BIRTH_DT_TM, "MM/DD/YY;;D") )
