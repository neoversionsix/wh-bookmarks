The CALL CLEAR command clears any portion of a display or the whole display.

Note

The CALL CLEAR command is only valid in the back-end screen application programs that use the Paint command. This type of program was widely used in Cerner Classic, but in general, it is only used for utility programs in Millennium Platform.

Syntax

CALL CLEAR (begin_row,begin_col[,clear_length])

Metasymbols

begin_col

The beginning column number (1 to 80 for 80-column mode and from 1 to 132 for 132-column mode) on the screen where the reverse-video box, scroll window, line, box, or text will be displayed.

begin_row

The beginning row number (1 to 24) on the screen where the box, reverse-video box, text, line, or scroll window will be displayed.

clear_length

The number of columns from the beginning position to clear the display. For example, if begin_row is 5 and clear_length is 20, then all data from column 5 to 25 will be cleared for that line. If length is not used, then all data from the starting row and column will be cleared to the end of the display.

Examples

The following examples illustrate different ways to use CALL CLEAR:

To clear the first line of a display for an 80-column display width, enter the following command:

CALL CLEAR (1,1,80)

To clear a portion of the fifth line of a display for an 80-column display width (positions 5 through 25 are cleared while positions 1 through 24 and positions 26 through 80 remain), enter the following command:

CALL CLEAR (5,5,20)

To clear everything on a display starting at Line 5 to the bottom of the display, enter the following command:

CALL CLEAR (5,1)

To clear the whole display, enter:

CALL CLEAR (1,1)

To clear a box in the display from row 10, column 20 to row 15, column 50 (while the rest of the display remains), enter the following command:

SET NUM = 10
WHILE (NUM <= 15)
CALL CLEAR (NUM,20,30)
SET NUM=NUM+1
ENDWHILE
