This function prevents trimming trailing spaces from a string that is assigned to a variable character data type variable or record structure item.

Syntax

NOTRIM(expr)

Metasymbol

expr
A character value.

Examples

The following examples demonstrate the use of the NOTRIM( ) function.

DROP PROGRAM NO_TRIM_EX GO
CREATE PROGRAM NO_TRIM_EX
RECORD REC(1 FLD1 = VC)		;auto trim trailing spaces by default
SET REC->FLD1="ABC   "
CALL ECHO(BUILD('SIZE OF REC->FLD1=',SIZE(REC->FLD1)))	;size of rec->fld1=3
SET REC->FLD1=TRIM("ABC   ")
CALL ECHO(BUILD('SIZE OF REC->FLD1=',SIZE(REC->FLD1))) 	;size of rec->fld1=3
SET REC->FLD1=NOTRIM("ABC   ")
CALL ECHO(BUILD('SIZE OF REC->FLD1=',SIZE(REC->FLD1))) 	;size of rec->fld1=6

declare vc_var = vc with protect 
set vc_var = "abc  " 
call echo(concat(vc_var,"*")) ;echoes abc*

set vc_var = notrim("abc  ") 
call echo(concat(vc_var,"*")) ;echoes abc  *

END
GO
