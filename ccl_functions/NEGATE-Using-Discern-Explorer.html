Performs a Boolean negation on a Boolean.

Syntax

NEGATE(Boolean_expression)

Metasymbol

boolean_expression
A Boolean expression

Examples

In this example, the result is 0 or false. returns false if the Boolean_expression is true else it returns true.

SET X=TRUE
SET X=NEGATE(X)
