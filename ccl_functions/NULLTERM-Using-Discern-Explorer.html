Null terminates a character expression. Any trailing spaces at the end of the character expression is trimmed. A binary zero is added to the end of the character string.

Syntax

NULLTERM(expression, [option])

Metasymbol

expression

A character value.

option

1: Include the null terminator character (binary zero) in the length of the string.

Examples

The following example includes the null terminator:

DECLARE NT_NAME = VC
SET NT_NAME = NULLTERM(P.NAME_FULL_FORMATTED,1)

DECLARE MEANING_VAR = C12
SET MEANING_VAR = "COMBINED"
DECLARE COMBINED_VAR = F8
SET COMBINED_VAR = UAR_GET_CODE_BY("MEANING",48,NULLTERM(MEANING_VAR))
