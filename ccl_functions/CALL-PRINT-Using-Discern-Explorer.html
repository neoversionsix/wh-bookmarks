The CALL PRINT command prints the ASCII result of an expression. For the BUILD( ), CALCPOS( ), and TRIM( ) functions, the current position is left at the end of the variable-length result. This command is valid only in the Reportwriter section of a program.

Syntax

CALL PRINT (expression)

Metasymbol

expression
An expression is the highest-level metasymbol among constants, field names, variables, and so on. The following expressions are valid:

alias.field_name
array_name [dimension]
array_name [dimension1, dimension2]
array_name [dimension1, dimension2, dimension3]
constant
expression arithmetic_operator expression
-expression
(expression)
function(expression{,expression})
parameter
record_name{->field_name}
table_name.field_name
variable

Note

Square brackets are part of the array_name syntax; they do not indicate an optional part of the syntax.

Examples

The following example prints the contents of the variable VAR1 embedded in angle brackets, with any trailing and leading spaces removed:

CALL PRINT (CONCAT("<",TRIM(VAR1),">"))
{"serverDuration": 555, "requestCorrelationId": "af74f61ad694a97f"}
