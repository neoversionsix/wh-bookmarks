The FOR command is used to process a block of Discern Explorer commands multiple times. The FOR command sets the variable equal to the value of integer1, executes the for_items, then increments the variable by the increment value if the BY increment is used. If the BY increment is not used, the variable is incremented by one. After the increment is performed, the FOR command checks to see if the variable is greater than the value of integer2. If the variable is not greater than the value of integer2, the for_items are executed again and the variable is incremented. The for_items continue to be executed, and the variable continues to be incremented until the variable becomes greater than the value of integer2. The FOR command can be used instead of a manual counter with a WHILE command.

Syntax

FOR (variable = integer1 TO integer2 [BY increment])
	{for_item}
ENDFOR

Metasymbols

for_item

A command or instruction performed by a FOR statement. Valid FOR items are commands and report items. A command is a Discern Explorer instruction that can be executed by itself (for example, BROWSE, CREATE TABLE, DROP PROGRAM, and SELECT) or included as part of another instruction (for example, CASE and WHILE).

A report item is a printable data element, calculation, or conditional used within a Reportwriter clause. The following are valid report items:

variable

integer1

A whole number that is the beginning value to be assigned to the variable by the FOR loop.

integer2

A whole number that is the ending value to be assigned to the variable by the FOR loop.

increment

A whole number that is used to increment the variable in the FOR loop.

variable

A variable name. If the variable name does not already exist, it is created by default.

Before Discern Explorer versions 8.14.1 and 9.02.1, namespaced variables were not allowed for the integer1, integer2, increment, or variable metasymbols. Discern Explorer versions 8.14.1 and 9.02.1 began support for namespaced variables for the integer1, integer2, increment, or variable metasymbols.

Examples

The following examples illustrate different ways to use FOR:

In the below example, the FOR loop displays the word TEXT in column five on the row indicated by the variable NUM. The word is displayed a total of ten times, and it is followed by an integer from one to 10 based on the value in NUM.

FOR (NUM = 1 TO 10)
	CALL TEXT (NUM, 5, BUILD ("TEXT", NUM))
ENDFOR

In the below example, the FOR loop uses an increment of two. The loop displays the word TEXT in column 5 five times on the row indicated by NUM.

FOR (NUM = 1 TO 10 BY 2)
	CALL TEXT (NUM, 5, BUILD ("TEXT", NUM))
ENDFOR
