The FINDSTRING( ) function allows you to return the position at which the search_expression begins in the source_expression. The start_pos can be used to perform a search for the next occurrence of the search text. The FINDSTRING( ) function is case-sensitive, therefore only like characters are found. When no matches are found, a zero is returned.

Syntax

FINDSTRING(search_expression,source_expression[,start_pos[,find_mode]])

Metasymbols

search_expression

The text expression to be searched for.

Use NOPATSTRING() around the search_expression when the source_expression is an RDBMS field and finding a literal character that can also be treated as a wildcard character.

source_expression 

The source text in which the search expression should be located.

start_pos

The position in the source expression where the search should begin. The default is 1.

find_mode 

The number representing the find mode.   

Mode

Description

0

Find first occurrence

1

Find last occurrence

Examples

In this example, LAST is set to 10.

LAST = FINDSTRING("xxx","abcxxxdefxxx",1,1)

The following example sets the variable POS equal to 4:

 SET POS = FINDSTRING("|", "abc|xyz")

The following example uses the NOPATSTRING() function around the search_expression when searching for A*C in the l.long_text field.

if(findstring(nopatstring("A*C"),l.long_text,1,0) > 0) 
	col 0 "Found" 
else 
	col 0 "Not Found"
endif
