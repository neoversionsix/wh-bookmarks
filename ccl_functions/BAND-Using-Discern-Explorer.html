The BAND( ) function returns an integer expression that is the result of a bit_number to bit_number AND comparison of two base-2 integers. The bit_number is the position of a bit in a field.

Syntax

BAND(integer,integer)

Metasymbols

integer
Any whole number.

The key to understanding how the BAND( ) function works is by converting the two numbers being compared to base-2 numbers first.

In base-2, numbers are represented by placing a zero or one in a bit number position and a subscript value of 2. In the base-10 system, the number base in which we usually count, we count from one to nine, carry the one, and start over at zero. The below examples display conversions from base-10 to base-2.

The BAND( ) function identifies if converting both integers to base-2 results in a value of 1 being displayed in both bit_number positions. If a value of 1 is displayed, then the result of the BAND( ) function also has a value of 1 displayed in the corresponding bit_number position.

Examples

The following example displays the first number as three and the second number as five:

RESULT=BAND(3,5)

310=000000112

510=000001012

Looking from right to left, you can see that 000000112 has a one in the zero and first bit_number positions and that 000001012 has a one in the zero and second bit_number positions. Thus the RESULT value has a one in the zero bit_number position.

In base-2, the following result is displayed:

RESULT=000000012

Converting this back to base-10, the following result is displayed:

RESULT=1

The following example displays the integers of 12 and 10:

RESULT=BAND(12,10)

1210=000011002

1010=000010102

RESULT=000010002

When you convert this back to base-10, the following result is displayed:

RESULT=8
