SET CURALIAS provides for a shortcut to access record structure items. A record alias can only be used in a script, and is only recognized in with the "->" connect symbol.

Syntax

SET CURALIAS [=] [namespace::]alias_name  [OFF]  record_path

Metasymbols

alias_name

A chosen name for the alias

namespace

Namespace allows creation of the alias in a specific namespace. If not specified, the default namespace is PUBLIC. Namespace is available with Discern Explorer versions 8.6.0 and higher.

record_path

A record structure member

Current limitations:

  1. By default the SET CURALIAS cannot be changed once set in a program. The Set Modify CurAliasReUse command can be used to override the default.
  2. The curalias variable is only recognized for the case curalias_name->field_name or namespace::curalias_name->field_name.
  3. Beginning with Discern Explorer versions 8.14.0 and 9.02.0, CURALIAS can use the name of an existing record structure as the alias_name, provided that SET MODIFY CURALIASREUSE is enabled.

Examples

In the example below, R1 is set to REPLY->QUAL[CNT] and R2 is set to REPLY->QUAL[CNT]->QUAL2[CNT2]. You can then use R1 to access any level 2 record structure members and R2 to access any level 3 record structure members.

DROP PROGRAM TEST GO
CREATE PROGRAM TEST

RECORD REPLY(
       1 QUAL[*]
            2   FLD1 = I4
            2   QUAL2[*]
                3   FLD2 = I4
            2   FLD3 = I4
      )
SET CURALIAS = R1 REPLY->QUAL[CNT]
SET CURALIAS = R2 REPLY->QUAL[CNT]->QUAL2[CNT2]
SET STAT = ALTERLIST(REPLY->QUAL,3)
FOR (CNT=1 TO SIZE(REPLY->QUAL,5))
	SET STAT = ALTERLIST(R1->QUAL2,2)
	SET R1->FLD1 = CNT
	CALL ECHO(BUILD("FLD1=", R1->FLD1))
	FOR(CNT2=1 TO SIZE(R1->QUAL2,5))
		SET R2->FLD2 = CNT2
		CALL ECHO(BUILD("FLD2=",R2->FLD2))
	ENDFOR
	SET R1->FLD3 = 100
	CALL ECHO(BUILD("FLD3=",R1->FLD3))
ENDFOR
SET CURALIAS R1 OFF  ;CLEAR THE ALIAS NOW
SET CURALIAS R2 OFF  ;CLEAR THE ALIAS NOW
END GO

In the following example, SR1::R is set to SPACE1::REC->LIST[LCNT] and R is set to REC->LIST[LCNT]:

drop program curalias_namespace go
create program curalias_namespace  

record space1::rec (
  1 list [*]  
    2 id = i4
) 

record rec (
  1 list [*]  
    2 id = i4
) 

set	stat = alterlist(rec->list,5)
set	stat = alterlist(space1::rec->list,5)

for(cnt = 1 to 5)
	set rec->list[cnt].id  = cnt
	set space1::rec->list[cnt].id  = cnt *1000
endfor

call echorecord(rec)
call echorecord(space1::rec)

set curalias  sr1::r space1::rec->list[lcnt]
set curalias  r rec->list[lcnt]

set lcnt = 1

call echo(sr1::r->id)  ;will echo 1000
call echo(r->id)       ;will echo 1

set curnamespace = "space1"
call echo(r->id)  ;will echo 1000

set curnamespace = "PUBLIC"
call echo(r->id)  ;will echo 1

end go
