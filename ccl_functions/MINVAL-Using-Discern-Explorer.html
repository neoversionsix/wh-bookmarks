The MINVAL() function allows you to return the smallest expression from the list of expressions.

Syntax

MINVAL(numeric_expr,numeric_expr{,numeric_expr})

Metasymbol

numeric_expr
An expression that is an integer or a real number.

Examples

In the following example, result is set equal to 20.

drop program hhtest go
create program hhtest
set first = 71
set second = 20
set third = 51
set fourth = 40
set fifth = 37
select
	result = minval(first,second,third,fourth,fifth)
from dummyt
end
