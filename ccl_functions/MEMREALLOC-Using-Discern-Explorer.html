MEMREALLOC () redefines a scalar variable or one-dimensional array and reallocates the dynamic memory in which the array or variable is stored. This function allows you to resize a dynamic memory array by increasing or decreasing the number of elements previously defined using the MEMALLOC( ) function.

Syntax

MEMREALLOC(array_name,integer,"data_type")

Metasymbols

array_name
The name of the one-dimensional array or scalar variable to redefine.

data_type
Defines the characteristics of a column.

Type

Description

Cwidth

Character string and length

F4

Single-precision float

F8

Double-precision float

[U]I1

Unsigned/signed 1-byte integer

[U]I2

Unsigned/signed 2-byte integer

[U]I4

Unsigned/signed 4-byte integer

[U]I8

Unsigned/signed 8-byte integer

[R]DNwidth

Reverse/normal numeric date

[R]Nwidth[.precision][sign]

Reverse/normal unsigned/signed numeric

[R]TNwidth

Reverse/normal numeric time

[R][U]DPpack

Reverse/normal unsigned/signed packed date

[R][U]Ppack[.precision]

Reverse/normal unsigned/signed packed

[R][U]TPpack

Reverse/normal unsigned/signed packed time

VCwidth

Variable-length character string

-

-

pack

Total number of packed digits.

precision

Number of digits to the right of the implied decimal place.

sign

For COBOL data types where usage is DISPLAY, this type describes whether a sign accompanies the data and if so,
whether it trails or leads the data, and whether it is separate from the data. The syntax is:([TRAIL/LEAD] [,SEPARATE]).

width

Internal storage width in bytes.

integer
The number of elements to define in the array. It is also described as the number of occurs or positions in the array.

Examples

The below example creates a one-dimensional dynamic memory array called ALIAS that has 10 elements, each of which is 15-character column. STAT is a variable whose value is 1 if the dynamic memory array ALIAS is created or redefined successfully, and 0 if the dynamic memory array ALIAS is not created or redefined successfully.

drop program hhtest go
create program hhtest
set stat  = memalloc(alias,10,"c15")
select
	v_alias = substring(1,15,p.ALIAS)
from person_alias p
head report
	cnt = 0
	xx =initarray(alias," ")
	cntx = 0
	incr  = 10
detail
	if (cnt = incr)
		incr = incr + 10
		stat = memrealloc(alias,incr,"c15")
	endif
	cnt = cnt + 1
	alias[cnt] = v_alias
foot report
	for (lcnt = 1 to cnt)
	alias[lcnt]
	row + 1
	endfor
with maxrec = 100
end
go

The following example uses MEMREALLOC() to redefine the size of the OUTBUF variable:

select lb.long_blob_id
	,lb.long_blob
from long_blob lb
plan lb where lb.long_blob_id = $ID

    outbuf = ""
    imageDataSize = blobgetlen(lb.long_blob)  ;fetch full length of blob
    stat = MEMREALLOC(outbuf,1,build("C",imageDataSize)) ;resize using full length to fetch with one blobget call
    fullblob = blobget(outbuf,0,lb.long_blob)
with rdbarrayfetch=1
