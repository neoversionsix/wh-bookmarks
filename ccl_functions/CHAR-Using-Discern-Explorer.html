Returns a character expression from an ASCII ordinal integer expression. CHAR( ) is the converse of the ICHAR( ) function. It returns an ASCII character for an ordinal number.

Syntax

CHAR(integer_expr)

Metasymbol

integer_expr
An an integer representing an ASCII ordinal character. An internet search for "Complete ASCII table" without the quotes will return numerous links to tables that can be used to cross reference the ASCII ordinal values to their character equivalents.

Examples

In this example, X is set to A.

X = CHAR(65)

In this example, LF is set to the linefeed character.

LF = CHAR(10)

In this example, CR is set to the carriage return character.

CR = CHAR(13)
