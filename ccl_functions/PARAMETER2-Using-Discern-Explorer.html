The Parameter2( ) function returns the parameter number for the $param_name. It is often used within the Parameter( ) function, which requires the parameter number. The Parameter2() function was introduced in version 8.7.0 of Discern Explorer.

Syntax

Parameter2($param_name)

Metasymbols

$param_name

param_name is the name assigned to a prompt using the With option of the Prompt clause within the Create Program command.

Examples

In the below example, the variable gender_num is set to 2 because gender is the name assigned to the second prompt by the With option of the Prompt clause. If multiple values are passed in an item list from the gender control on the prompt form, the variable lcheck is set to L. If a single value is passed, lcheck is set to I, F, or C based on the data type of the value.

drop program ccl_test_parameter2 go
create program ccl_test_parameter2

prompt 
	"Output to File/Printer/MINE" = "MINE"
	,"Select the Genders" = 0 

with OUTDEV, gender

;determine the prompt number of the prompt named "gender"
set gender_num = parameter2($gender)

;check to see if multiple values were selected at the $gender prompt
set lcheck = substring(1,1,reflect(parameter(parameter2($gender),0)))

call echo(gender_num)
call echo(lcheck)


end
go
