The MOVESTRING () function copies a character expression into another character expression and returns the number of characters that were copied. The first string expression, starting at the position indicated by source_start, is moved to the second string expression, starting at the position indicated by target_start. The number of characters moved is specified by length. The characters in the target_text are overwritten by the characters from the source_text. No change is made to the source_text.

An error message is generated if you try to move an expression to a read only variable. The same message is generated if you set a variable to an expression in the Reportwriter section.

Syntax

MOVESTRING(source_text,source_start,target_text,target_start,length)

Metasymbols

source_text
The text expression to be copied.

source_start
An integer that defines the beginning position in the source_text.

target_text
The text expression to which the source text is copied.

target_start
An integer that defines the beginning position in the target_text.

length
An integer that defines the number of characters to copy.


Examples

In the example below, LEN is set to 5, and five asterisks are copied into the variable VARA beginning at position 10. The characters in positions 10 through 15 of VARA are overwritten with the asterisks.

LEN = MOVESTRING("**********",1,VARA,10,5)
