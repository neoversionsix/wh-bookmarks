The SUBSTRING ( ) function returns a specified number of characters from a text expression. A beginning position is specified, then a number of characters to extract, and finally a text expression from which to extract the characters is specified.

Syntax

SUBSTRING(start,length,text_expr)

Metasymbols

start

An integer that defines a position in the text expression. The character that is in this position of the text expression is the first character returned. The first position of the text expression is 1.

length

An integer that defines the number of characters to extract from the text expression.

text_expr

A text expression.

Examples

In the following example, SVAR is set to ABCD:

SET SVAR = SUBSTRING(1,4,"ABCDEFGHI")

In the following example, SVAR is set to EFG:

SET SVAR = SUBSTRING(5,3,"ABCDEFGHI")

Note

A start position less than one is treated as starting at position one ( 1 ). Also, if the length specified extends beyond the length of text_expr, the resulting string length is padded with spaces. These behaviors are only obvious when the SUBSTRING( ) function is wrapped in a NOTRIM( ) function.

declare t = vc go
set s = "ABCD" go
set t = notrim(substring(-1,3,s)) go
call echo(concat("*",t,"*")) go

set t = notrim(substring(0,3,s)) go
call echo(concat("*",t,"*")) go

set t = notrim(substring(5,3,s)) go
call echo(concat("*",t,"*")) go

set t = notrim(substring(2,3,s)) go
call echo(concat("*",t,"*")) go

set t = notrim(substring(2,9,s)) go
call echo(concat("*",t,"*")) go
