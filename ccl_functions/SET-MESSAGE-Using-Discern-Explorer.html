The SET MESSAGE command controls whether Discern Explorer error messages can be passed to the display when Discern Explorer programs using screen generation commands are executed. The default for servers is SET MESSAGE NOINFORMATION (do not display debug information). The default for an interactive Discern Explorer session is SET MESSAGE INFORMATION. You can define how long a message remains on the screen or use the -1 option to suppress messages. Messages are displayed on Line 24 of the screen. The SET MESSAGE WINDOW option is used to enter Screen Application mode, which allows you to use parameters ($).

Enter GO at the end of the command to activate the message option throughout a Discern Explorer session. If you omit GO, the SET MESSAGE command option is only active during the execution of the program that called it, and it must be embedded within an executable program (between the CREATE PROGRAM and END commands).

Syntax

SET MESSAGE = message_option [GO]

Metasymbol

message_option determines how Discern Explorer updates the display with Discern Explorer error messages when a program is executing. The following values are valid:

ValueDescription

INFORMATION

Debug information is written to the Discern Explorer Run-Time Library (RTL) log file.

NOINFORMATION

Debug information is not written to the Discern Explorer Run-Time Library (RTL) log file.

Seconds

The number of seconds to allow any Discern Explorer error message to remain on the display during the execution of a Discern Explorer screen application program that is enabled by the Paint clause in the CREATE PROGRAM or SET MESSAGE WINDOW command. The default is 10 seconds or until the user presses a key to continue. Use -1 to suppress error messages. (You cannot use zero).

WINDOW

Begin a paint application without clearing the screen.

NOWINDOW

End a paint application.

Examples

The below examples illustrate different ways to use SET MESSAGE:

To allow messages to be displayed with minimal information when any Discern Explorer program is executing during a Discern Explorer session, enter the below command. The message remains for 20 seconds or until the user presses any key to continue.

SET MESSAGE = 20 GO

The below example suppresses Discern Explorer error messages during a Discern Explorer screen application. This command must be used after Paint in the CREATE PROGRAM or SET MESSAGE WINDOW command:

SET MESSAGE = -1
