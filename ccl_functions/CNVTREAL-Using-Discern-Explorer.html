CNVTREAL () returns a result of type Real that is a conversion of a non-real expression to a real expression.

Syntax

CNVTREAL(nonreal_expr)

Metasymbol

nonreal_expr
An expression that is a non-real number.

Examples

The following example sets RMONEY equal to 100.45:

RMONEY = CNVTREAL("100.45")
