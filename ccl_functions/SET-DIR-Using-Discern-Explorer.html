The SET DIR command changes your default directory in a Discern Explorer session. The SET DIR command temporarily reassigns the CCLUSERDIR logical to point to the directory you have identified during this session or until you use the SET DIR command again. CCLUSERDIR is the default directory where extract and input files reside. SET DIR is invalid when executing in a server process. It is only valid in interactive command-line sessions.

Use GO at the end of the command when you are outside of an executable program (for example, when you are at the Discern Explorer command line).

Syntax

SET DIR [=] "string" [GO]

Metasymbol

The string must be embedded in quotation marks and cannot exceed 31 characters. Use the SET LOGICAL command if this limit is exceeded.

Examples

The following example changes your default directory to PATH1:[USER.XYZ]:

SET DIR "PATH1:[USER.XYZ]" GO
