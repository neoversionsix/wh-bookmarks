The SET HOME command identifies what action the program should take when a user presses HOME. The default is OFF; no action is taken when HOME is pressed.

Use GO at the end of the SET HOME command to make it valid throughout a Discern Explorer session. If you do not use GO, the SET HOME command is valid only during the execution of the program that called it and must be embedded in an executable program (between the CREATE PROGRAM and END commands).

Syntax

SET HOME = home_option [GO]

Metasymbol

home_option

Determines the action to be taken when the Home key is pressed. The following options are valid:

Option

Description

label

Branch to the set of Discern Explorer instructions represented by label.

OFF

Do not take any action when HOME is pressed.

Examples

The examples below illustrate different ways to use SET HOME.

To turn the HOME key off (so no action is taken when the key is pressed), enter the following command:

SET HOME = OFF

To tell Discern Explorer to branch to the label #CORRECT_YN when HOME is pressed in a Discern Explorer program, enter the following command:

SET HOME = CORRECT_YN

(The Discern Explorer instructions under the #CORRECT_YN label might give the user the ability to correct entries in fields on the screen.)
