The CALL VIDEO command changes characteristics of text displayed using the CALL BOX, CALL CLEAR, CALL LINE, and CALL TEXT commands. CALL VIDEO can change lines, boxes, and text to display as normal, blinking, reverse-underlined video, 80-column, or 132-column. For screen displays, normal mode is bold letters (intense) displayed in the default font and pitch for your terminal. The CALL VIDEO command typically is used in screen application programs to set the video mode back to normal before generating a report. All subsequent displays use this mode until another CALL VIDEO command is issued.

Note

The CALL VIDEO command is only valid in the back-end screen application programs that use the Paint command. This type of program was widely used in Cerner Classic, but in general, it is only used for utility programs in Millennium Platform.

Syntax

CALL VIDEO (video_option{video_option})

Metasymbol

video_option
The display mode to be used for the box, line, or text. The following values are valid:

Value

Description

B

Blink.

I

High intensity.

L

Low intensity.

N

Normal (no blinking, no underline, no reverse). Reset the display to high-intensity-only mode.

R

Reverse video.

S

Slender (80-column mode, the default). Using option N does not reset the screen to 80-column mode.

U

Underline.

W

Wide (132-column mode).

Examples

The following examples illustrate different ways to use CALL VIDEO:

To reset all blinking, reverse-video, or underlined boxes, text, or lines to normal mode, enter:

CALL VIDEO (N)

To set the display mode to blinking, enter:

CALL VIDEO (B)

To set the display to blinking, reverse-video, 132-column display, enter:

CALL VIDEO (BRW)
