WEEKDAY returns the integer day of the week component of an expression that is a Julian date or rdb_date_time. An integer from 0 to 6 is returned (with Sunday = 0).

Syntax

WEEKDAY(julian_date)
WEEKDAY(rdb_date_time)

Metasymbols

julian_date

An expression that is a date stored in Julian format.

rdb_date_time

A column that stores the date and time.

Examples

In this example, if the date in the E.DISCHG_DT_TM column equals Thursday, December 31, 1998, W1 is set to 4. If the current date is Tuesday, October 27, 1998, W2 is set to 2.

W1 = WEEKDAY(E.DISCHG_DT_TM)
W2 = WEEKDAY(CURDATE)
