The CALL BOX command draws a box on the terminal screen for a Discern ExplorerÂ screen application program.

Note

The CALL BOX command is only valid in the back-end screen application programs that use the Paint command. This type of program was widely used in Cerner Classic, but in general, it is only used for utility programs in Millennium Platform.

Syntax

CALL BOX (begin_row,begin_col,end_row,end_col)

Metasymbols

begin_col

The beginning column number (from 1 to 80 for 80-column mode and from 1 to 132 for 132-column mode) on the screen where the reverse-video box, scroll window, line, box, or text will be displayed.

begin_row

The beginning row number (from 1 to 24) on the screen where the box, reverse-video box, text, line, or scroll window will be displayed.

end_col

The ending column number (from 1 to 80 for 80-column mode and from 1 to 132 for 132-column mode) on the screen where the scroll window or box will be displayed.

end_row

The ending row number (from 1 to 24) on the screen where the scroll window or box will be displayed.

Examples

The following example draws a box on the terminal screen where the upper-left corner of the box is at row 5, column 5, and the lower-right corner of the box is row 20, column 50:

CALL BOX (5,5,20,50)
