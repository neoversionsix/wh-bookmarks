Returns the total length of a long, longraw, blob, or clob data type. The WITH RDBARRAYFETCH = 1 control option is implicitly set if it is not used explicitly. If the query that uses the BLOBGETLEN() performs a sort, the sort must be done at the RDBMS level. Sorting done at the Discern Explorer level is not allowed.

Syntax

BLOBGETLEN(Blob_Attribute)

Metasymbols

Blob_Attribute

A long, longraw, blob, or clob data type field or value.

Examples

select into "NL:"
	lb.long_blob_id
	,lb.long_blob  
from long_blob lb
plan lb where lb.long_blob_id = $ID
