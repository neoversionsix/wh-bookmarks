Returns an integer that is the remainder of the first integer divided by the second integer (mod_value).

Syntax

MOD(integer,mod_value)

Metasymbols

integer
Any whole number or an expression that is an integer.

mod_value
An integer or variable divided into an integer expression to produce a whole number remainder.

Examples

Drop program hhtest go
create program hhtest
set value = 10
select
	result = cnvtint(substring(1,12,C.RESULT_VAL))
from clinical_event c
detail
col 5 "RESULT:" COL 25 "RESULT CODE:"
ROW + 1
 rec_code = mod(RESULT,value)
 col 5 result col 25 rec_code
 row + 1
with maxrec = 50
end
