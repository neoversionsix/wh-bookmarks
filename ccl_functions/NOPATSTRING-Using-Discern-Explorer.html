NOPATSTRING () skips pattern-matching in the expression.

Syntax

NOPATSTRING(expression)

Metasymbol

expression
A character data type value.

Examples

The below example qualifies rows where Result_Val is equal to the literal string "DQB1*0201". Without the NOPATSTRING() the code in the example qualifies rows where Resutl_Val begins with "DQB1", followed by any number of characters, and ends with "0201".

SELECT INTO "NL:"
	C.RESULT_VAL
FROM
	CLINICAL_EVENT   C
WHERE C.RESULT_VAL = NOPATSTRING("DQB1*0201")
{"serverDuration": 509, "requestCorrelationId": "67eba3b5b4c02244"}
