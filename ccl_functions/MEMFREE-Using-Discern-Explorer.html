The MEMFREE( ) function allows you to release the dynamic memory that has been allocated for a one-dimensional array using the MEMALLOC() function. The MEMFREE( ) function is analogous to the FREE SET command.

Syntax

MEMFREE(array_name)

Metasymbol

array_name
The identifier given to a series of elements arranged in rows and columns. Discern Explorer allows you to define one-, two-, and three-dimensional arrays.

Examples

The following example frees the dynamic memory allocated for the array V1, and frees the variable from the symbol table.

STAT = MEMFREE(V1)
