The CALL ECHO command echoes an expression to the displayer, log file, or listing file. The CURECHO variable is used to compare to the echo_level. If the echo_level is less than CURECHO, the CALL ECHO command is not performed. CURECHO is reset to 0 at the start of a parent ekmodule or program. If ECHO trace is not active, CALL ECHO commands are ignored.Â Discern Explorer version 8.12.6 introduces the ability to use ALL as an array index to iterate and echo all positions of an array.

Syntax

CALL ECHO (echo_expression [,print_indicator [,echo_level]] )

Metasymbols

echo_expression
An expression to be echoed.

print_indicator
A numeric value of 0 or 1 used to determine if a new line character is printed. A value of 0 overrides the default of 1 and allows multiple CALL ECHO's to display on the same line. The print_indicator is optional.

echo_level
A numeric value. The expression is only echoed if echo_level is greater than or equal to CURECHO.

Examples

The following example illustrates different ways to use CALL ECHO:

SET CURECHO = 3	; change level from 0 to 3
CALL ECHO("TEST1",1,4)	; issue the call echo
CALL ECHO("TEST1",1,2)	; ignore the call echo
CALL ECHO("TEST")
CALL ECHO("PROC SELECT")
CALL ECHO(BUILD("NUM_QUAL",CURQUAL) )

The following example defines one and two dimensional arrays, and uses the ALL option to echo all positions of the array:

declare 1d_array[3] = c1
set 1d_array[1] = "A"
set 1d_array[2] = "B"
set 1d_array[3] = "C"
call echo(1d_array) ;since no array index is specified defaults to [1]
call echo(1d_array[2])   ;echoes [2]
call echo(1d_array[all]) ;echoes all positions of 1d_array 

declare 2d_array[3,2] = c3
set 2d_array[1,1] = "1-1"
set 2d_array[1,2] = "1-2"
set 2d_array[2,1] = "2-1"
set 2d_array[2,2] = "2-2"
set 2d_array[3,1] = "3-1"
set 2d_array[3,2] = "3-2"
call echo(2d_array[1,1]) ;echoes 1-1
call echo(2d_array[1,all])   ;echoes 1-1 and 1-2
call echo(2d_array[all,all]) ;1-1, 1-2, 2-1, 2-2, 3-1, and 3-2
