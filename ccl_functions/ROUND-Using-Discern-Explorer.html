Round () rounds a real expression to the number of positions to the right of the decimal specified by the second metasymbol. Positive numbers with a trailing digit of 5 to the right of the decimal are rounded up. Before Discern Explorer versions 8.13.3 and 9.01.3, negative numbers with a trailing digit of 5 to the right of the decimal were rounded up (closer to zero). Beginning withÂ Discern Explorer versions 8.13.3 and 9.01.3, negative numbers with a trailing digit of 5 to the right of the decimal are rounded down (away from zero) to match the behavior of rounding with positive numbers.

Syntax

ROUND(real_expr,precision)

Metasymbols

precision
An integer representing the number of positions to the right of the decimal.

real_expr
An expression that is a real number.

Examples

In the following example, round_var is set to 11.0:

SET Round_Var = ROUND(10.54678,0)

In the following example, round_var is set to 10.5:

SET Round_Var = ROUND(10.54678,1)

In the following example, round_var is set to 10.55:

SET Round_Var = ROUND(10.54678,2)

The following example echoes 31.02 and 31.03:

CALL ECHO(ROUND(31.015,2)) GO
CALL ECHO(ROUND(31.025,2)) GO

Before Discern Explorer versions 8.13.3 and 9.01.3, the following example echoed 31.02 and -31.01. In Discern Explorer versions 8.13.3, 9.01.3, and later, the following example echoes 31.02 and -31.02:

CALL ECHO(ROUND(31.015,2)) GO
CALL ECHO(ROUND(-31.015,2)) GO
