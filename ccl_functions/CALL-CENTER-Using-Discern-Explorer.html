The CALL CENTER command centers a textual expression in a beginning and ending column in the Report Page row. CALL CENTER can be used only in the Reportwriter section of a program, and is intended for use when creating plain text output. See CENTER Using Discern Explorer for information about centering in DIO.

Syntax

CALL CENTER(expression,begin_col,end_col)

Metasymbols

Begin_col
The beginning column number (from 1 to 80 for 80-column mode, and from 1 to 132 for 132-column mode) on the screen where the reverse-video box, scroll window, line, box, or text is displayed.

End_col
The ending column number (from 1 to 80 for 80-column mode, and from 1 to 132 for 132-column mode) on the screen where the scroll window or box is displayed.

Expression
An expression is the highest-level metasymbol among constants, field names, variables, and so on. The following expressions are valid:

alias.field_name
array_name [dimension]
array_name [dimension1, dimension2]
array_name [dimension1, dimension2, dimension3]
constant
expression arithmetic_operator expression
-expression
(expression)
function(expression{,expression})
parameter
record_name{->field_name}
table_name.field_name
variable

Note

Square brackets are part of the array_name syntax; they do not indicate an optional part of the syntax.

Examples

The following example centers the contents of the variable CURRENT_NAME on a 132-column page:

CALL CENTER (CURRENT_NAME,1,132)
