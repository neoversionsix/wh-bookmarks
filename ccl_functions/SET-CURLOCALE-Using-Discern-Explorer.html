The SET CURLOCALE command allows you to change the value of the locale name. This command is intended for Oracle internal use and predefines locale settings in the CCLSTARTUP_LOCALE.PRG program. SET CURLOCALE must not be executed in any other script, program, or process that is executed by a server. SET CURLOCALE can be used for testing in an interactive back-end session.

Syntax

SET CURLOCALE locale_name_string locale_value_string [GO]

Metasymbol

local_name_string
Standard locale names.

local_value_string
A string value assigned to a standard locale name. This gives Oracle the ability to flex by language.

Examples

The following example sets the "DOLLAR" locale name to "DM". These values are set in the start-up file to flex by language and should not be modified.

SET CURLOCALE  "DOLLAR" "DM" GO
