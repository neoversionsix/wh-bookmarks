The SET SPOOL command prints a file from in a program or at the Discern Explorer command line in any session. Use GO at the end of the command to make it valid throughout a Discern Explorer session. If you do not use GO, the SET SPOOL command is valid only during the execution of the program that called it, and it must be embedded in an executable program (between the CREATE PROGRAM and END commands).

Syntax

SET SPOOL [=] string printer_name [WITH spool_option {, spool_option}] [GO]

Metasymbols

string
The full path and file name of the file to be printed. Standard operating system conventions apply, so you can leave off the node, disk drive, or directory name if you are already pointing to that place. The string must be embedded in quotation marks.

printer_name
The logical name of the printer queue to which the file will be spooled (for example, LP01 or LP02).

spool_option
Determines features related to the file printing process. The following options are valid:

Option

Description

CALLPRG = string|CURPROG

Append the current program name or defined string as a tag to the print job.  If omitted, the default is to apply the CURPROG to the print job. Introduced in version 8.9.4 of Discern Explorer

COMPRESS

Prints in compressed format.

NOCOMPRESS

Prints in normal noncompressed format.

COPY = num_copies

Defines the number of copies to be printed. 

DELETED

Deletes a file automatically after it is printed and also <file>.cpc if one exists.

NODELETED

Does not delete a file after printing. This is the default.

DIO = diomode

Allows printing a file that uses DIO control options and DIO Commands

DUPLEX

Two-sided printing. Prints on both sides of the paper, from edge to edge (as in a book).

FILECASE

Discern Explorer version 8.12.0 introduced the FILECASE option. By default Discern Explorer converts the full path and file name to lowercase characters. Specifying FILECASE will retain the full path and file name with no conversion.

LANDSCAPE

Prints in landscape mode.

NOLANDSCAPE

Print in portrait mode.

NOTIFY

Displays a message when a file is printed.

NONOTIFY

Suppress the message displayed when a file is printed. This is the default.

MEDIA = media_name

Specify optional media name.

PRINT = setup_module

Allows the selection of a print tray. Examples: TRAY1, TRAY2, TRAY3, TRAY4, TRAY5, PSTRAY1, PSTRAY2, PSTRAY3, PSTRAY4, PSTRAY5, A4TRAY, LETTERTRAY, LEGALTRAY, LEDGERTRAY, CUSTOM1, CUSTOM2, CUSTOM3, CUSTOM4, and CUSTOM5.

SIMPLEX

One-sided printing.

TUMBLE

Two-sided printing. Prints on both sides of the paper, end over end (as in a flip chart).

USER = username

Specify optional username. Discern Explorer versions 8.16.0  and 9.04.0 introduce an enhancement to issue a warning message if the username contains invalid characters. Examples of invalid characters are a blank, /, \\, ^, ", *,and |.

Examples

The following examples illustrate different ways to use SET SPOOL:

To print two copies of a file on the printer LP02 from the Discern Explorer command line, enter:

SET SPOOL = "CER_TEMP:PATIENTS.DAT" LP02 WITH COPY=2 GO

To print a file to the printer LP01 from in a Discern Explorer program, enter:

SET SPOOL "TURN_RPT.DAT" LP01
