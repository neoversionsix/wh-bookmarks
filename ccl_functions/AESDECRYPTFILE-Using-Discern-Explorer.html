The AESDECRYPTFILE( ) function decrypts a file that was encrypted using AESENCRYPTFILE().

Syntax

AESDECRYPTFILE(aeskeyid, infile, outfile, password [,mode])

Metasymbols

Aeskeyid
Integer value in the range of 1 - 5, must match the value that was used when the file was encrypted using AESENCRYPTFILE()

Infile
Name of encrypted file

Outfile
Name of decrypted file to create

Password
Nonblank password or blank password. If a nonblank password was used when the file was encrypted using AESENCRYPTFILE(), the same nonblank password must be used to decrypt the file using AESDECRYPTFILE().

Mode
Integer value - Limited to 0 (for base 10) or 1 (for base 16)

Examples

The following example creates a file named CCL_TEST.DAT using a SELECT INTO command. The AESENCRYPTFILE() function is then used to create file named CCL_ENCRYPT.DAT that will contain the encrypted data from the CCL_TEST.DAT file. The AESDECRYPTFILE() function is then used to create a file named CCL_DECRYPT.DAT that will contain the decrypted data from the CCL_ENCRYPT.DAT file.

SELECT INTO CCL_TEST
	P.PERSON_ID,
	NAME = SUBSTRING(1,40,P.NAME_FULL_FORMATTED)
FROM PERSON P	
WITH FORMAT, MAXREC = 10

SET STAT = AESENCRYPTFILE(1,"CCL_TEST.DAT","CCL_ENCRYPT.DAT","PWD",0)
SET STAT = AESDECRYPTFILE(1,"CCL_ENCRYPT.DAT","CCL_DECRYPT.DAT","PWD",0)
