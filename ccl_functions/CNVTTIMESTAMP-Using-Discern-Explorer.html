CNVTTIMESTAMP( ) converts a datetime and fractional seconds to a timestamp, which is a DM12 data type value that represents the current date, time, and fractional seconds. CNVTTIMESTAMP() was introduced in Discern Explorer version 8.6.3. Use the SYSTIMESTAMP variable for the current timestamp instead of using this function.

Syntax

SyntaxDescription
CNVTTIMESTAMP(datetime,fractsec)Convert a datetime and fractional seconds to a timestamp.
CNVTTIMESTAMP(timestamp)Convert a timestamp to a timestamp.
CNVTTIMESTAMP(datetime)Convert a datetime to a timestamp.
CNVTTIMESTAMP(timestamp_string)Convert a timestamp string to a timestamp.

Metasymbols

Metasymbols

Description

datetime

A column or value that stores the date and time.

fractsec

A nine-digit, right-justified number representing fractional seconds. For example, 1 is 000000001 and 123 is 000000123.

timestamp

A DM12 data type value that represents the current date, time, and fractional seconds.

timestamp_string

A string in "dd-mmm-yyyy hh:mm:ss.ccccccccc" format. The timestamp_string was introduced in Discern Explorer version 8.8.0. The fractional seconds value (ccccccccc) can be from 1 to 9 digits. Leading zeros in the fractional seconds value (ccccccccc) are ignored.

Examples

In the following example, TIMESTAMP_VAR is set equal to the timestamp value representing 1 hour, 45 minutes, 59 seconds, and 1 fractional second after noon on 05-APR-2012. TS_STR_2C is set equal to 05-APR-2012 13:45:59.10 and TS_STR_9C is set equal to 05-APR-2012 13:45:59.100000000

DECLARE TIMESTAMP_VAR = DM12    
DECLARE TS_STR_2C = VC
DECLARE TS_STR_9C = VC
SET TIMESTAMP_VAR = CNVTTIMESTAMP("05-APR-2012 13:45:59.1")
SET TS_STR_2C = FORMAT(TIMESTAMP_VAR,” DD-MMM-YYYY HH:MM:SS.CC;;D")
SET TS_STR_9C = FORMAT(TIMESTAMP_VAR,” DD-MMM-YYYY HH:MM:SS.CCCCCCCCC;;D")

In the following example, TIMESTAMP_VAR is set equal to the timestamp value representing 1 hour, 45 minutes, 59 seconds, and 123 fractional seconds after noon on 05-APR-2012. TS_STR_9C is set equal to 05-APR-2012 13:45:59.123000000 and TS_STR_2C is set equal to 05-APR-2012 13:45:59.12. Notice that the display template used when setting the TS_STR_2C variable is truncating the fractional seconds to two digits.

DECLARE TIMESTAMP_VAR = DM12    
DECLARE TS_STR_2C = VC
DECLARE TS_STR_9C = VC
SET TIMESTAMP_VAR = CNVTTIMESTAMP("05-APR-2012 13:45:59.123")
SET TS_STR_2C = FORMAT(TIMESTAMP_VAR, "DD-MMM-YYYY HH:MM:SS.CC;;D")
SET TS_STR_9C = FORMAT(TIMESTAMP_VAR, "DD-MMM-YYYY HH:MM:SS.CCCCCCCCC;;D")

If the following example is executed 4 hours, 26 minutes, and 49 seconds after noon on 12-Apr-2012, the TIMESTAMP_VAR is set to the timestamp value representing 4 hours, 26 minutes, 49 seconds, and 1 fractional second after noon on 12-Apr-2012. TS_STR_2C is set equal to 12-APR-2012 16:26:49.10 and TS_STR_9C is set equal to 12-APR-2012 16:26:49.100000000.

DECLARE TIMESTAMP_VAR = DM12    
DECLARE TS_STR_2C = VC
DECLARE TS_STR_9C = VC
SET TIMESTAMP_VAR = CNVTTIMESTAMP(CNVTDATETIME(CURDATE,CURTIME3),1)
SET TS_STR_2C = FORMAT(TIMESTAMP_VAR, "DD-MMM-YYYY HH:MM:SS.CC;;D")
SET TS_STR_9C = FORMAT(TIMESTAMP_VAR, "DD-MMM-YYYY HH:MM:SS.CCCCCCCCC;;D")

If the following example is executed 4 hours, 26 minutes, and 49 seconds after noon on 12-Apr-2012, the TIMESTAMP_VAR is set to the timestamp value representing 4 hours, 26 minutes, 49 seconds, and 123 fractional seconds after noon on 12-Apr-2012. TS_STR_9C is set equal to 12-APR-2012 16:26:49.123000000 and TS_STR_2C is set equal to 12-APR-2012 16:26:49.12. Notice that the display template used when setting the TS_STR_2C variable truncates the fractional seconds to two digits.

DECLARE TIMESTAMP_VAR = DM12    
DECLARE TS_STR_2C = VC
DECLARE TS_STR_9C = VC
SET TIMESTAMP_VAR = CNVTTIMESTAMP(CNVTDATETIME(CURDATE,CURTIME3),1)
SET TS_STR_2C = FORMAT(TIMESTAMP_VAR, "DD-MMM-YYYY HH:MM:SS.CC;;D")
SET TS_STR_9C = FORMAT(TIMESTAMP_VAR, "DD-MMM-YYYY HH:MM:SS.CCCCCCCCC;;D")
