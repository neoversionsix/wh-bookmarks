CNVTUPPER ( ) returns a result of type String that is a copy of an expression of type String with all lowercase characters converted to uppercase. The original expression is not modified. Use the CNVTLOWER()Â function to convert a string to lowercase.

Syntax

CNVTUPPER(text_expr)

Metasymbol

text_expr
A text expression.

Examples

In the following example, if P.NAME_FULL_FORMATTED = "Smith, Mary", then name = "SMITH, MARY":

NAME = CNVTUPPER(P.NAME_FULL_FORMATED)

The following example converts the value entered at the Name prompt to uppercase before comparing it to the P.NAME_LAST_KEY field:

WHERE P.NAME_LAST_KEY = CNVTUPPER($2)

The following example converts the value entered at the Name prompt to uppercase, and treats wildcard characters as pattern-matching strings.

WHERE P.NAME_LAST_KEY = PATSTRING( CNVTUPPER($2))
