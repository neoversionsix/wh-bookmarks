The EVALUATE2 () function returns the result of an IF() statement as a function call. Similar to the EVALUATE() function, the EVALUATE2() function is more powerful in that it allows relational operators to be used in the IF() clause.

Syntax

EVALUATE2(if_clause)

Metasymbols

if_clause
A valid IF() clause.

Examples

The following example uses the EVALUATE2() function to set an expression equal to "NURSE" or "DOC " based on multiple position codes:

DECLARE PATH_VAR = F
WITH CONSTANT(UAR_GET_CODE_BY("MEANING",88,"PATHOLOGIST"))
DECLARE RES_VAR = F8
WITH CONSTANT(UAR_GET_CODE_BY("MEANING",88,"RESIDENT"))
DECLARE LPN_VAR = F8 WITH CONSTANT(UAR_GET_CODE_BY("MEANING",88,"LPN"))
DECLARE RN_VAR = F8 WITH CONSTANT(UAR_GET_CODE_BY("MEANING",88,"RN"))

SELECT INTO $OUTDEV
P.PERSON_ID
, P.POSITION_CD
, P_POSITION_DISP = UAR_GET_CODE_DISPLAY(P.POSITION_CD)
, P.NAME_FULL_FORMATTED
, POSITION_TYPE = EVALUATE2(
IF(P.POSITION_CD = RN_VAR
OR P.POSITION_CD = LPN_VAR) "NURSE"
ELSEIF(P.POSITION_CD = RES_VAR
OR P.POSITION_CD = PATH_VAR) "DOC  "
ENDIF)
FROM PRSNL P
WHERE P.POSITION_CD IN (LPN_VAR, RN_VAR, RES_VAR, PATH_VAR)
