The CALL LINE command draws a vertical or horizontal line on a display. If the VERTICAL or XVERTICAL option is not used, a horizontal line is drawn by default. Horizontal lines are always drawn from left to right and vertical lines are always drawn from top to bottom; be sure to provide the starting coordinate (begin_row and begin_col). The XVERTICAL option draws an extended vertical line that contains a crossbar at each end. The XHORIZONTAL option draws an extended horizontal line.

Note

The CALL LINE command is only valid in the back-end screen application programs that use the Paint command. This type of program was widely used in Cerner Classic, but in general, it is only used for utility programs in Millennium Platform.

Syntax

CALL LINE (begin_row,begin_col,line_length[,direction])

Metasymbols

begin_col

The beginning column number (from 1 to 80 for 80-column mode and from 1 to 132 for 132-column mode) on the screen where the reverse-video box, scroll window, line, box, or text will be displayed.

begin_row

The beginning row number (from 1 to 24) on the screen where the box, reverse-video box, text, line, or scroll window will be displayed.

direction

Identifies how the line will be drawn. The following options are valid:

Property

Description

HORIZONTAL

Draw a line from left to right.

VERTICAL

Draw a line from top to bottom.

XHORIZONTAL

Draw a line from left to right with crossbars at each end.

XVERTICAL

Draw a line from top to bottom with crossbars at each end.

Line_length

The number of columns from the beginning position to draw the line. For example, if begin_row is 5, begin_col is 10, and length is 20, a line is drawn on Row 5 that is 20 bytes long, starting at Column 10 and ending at Column 30.

Examples

The following examples illustrate different ways to use CALL LINE:

To draw a horizontal line on Row 1 from Column 7 through Column 11, enter the following command:

CALL LINE (1,7,5)

To draw a vertical line from row 1, column 5 through row 10, column 5, enter the following command:

CALL LINE (1,5,10,VERTICAL)

To draw a box starting at row 2, column 2, with a horizontal line near the top on row 6, enter the following command:

CALL BOX(2,2,22,78)
CALL LINE(6,2,77,XHOR)
