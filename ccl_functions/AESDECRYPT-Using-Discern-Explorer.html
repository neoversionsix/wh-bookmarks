AESDECRYPT () decrypts a text string that was encrypted using the AESENCRYPT() function.

Syntax

AESDECRYPT(aeskeyid,inbuf,inbuflen[,mode])
AESDECRYPT(customkey,inbuf,inbuflen[,mode])

Metasymbols

aeskeyid
An integer value in the range of one to five. It must match the value that was used to encrypt the string using AESENCRYPT().

customkey
A 16-character string to use for the decryption. It must match the value that was used to encrypt the string using AESENCRYPT().

inbuf
The encrypted string that was created using the AESENCRYPT() function.

inbuflen
An integer value that represents the length of the character string to be decrypted.

mode
An integer value limited to 0 (for base-10) or 1 (for base-16)

Examples

In the below example, the decrypt_var is set equal to ABC12345xyz.

SET ENCRYPT_VAR = AESENCRYPT(1,"ABC12345xyz",TEXTLEN("ABC12345XYZ"),0) 
SET DECRYPT_VAR = AESDECRYPT(1, ENCRYPT_VAR,TEXTLEN(ENCRYPT_VAR),0)

In the below example, the decrypt_var is set equal to ABC12345xyz.

SET ENCRYPT_VAR = 	AESENCRYPT("My16CharacterKey","ABC12345xyz",TEXTLEN("ABC12345XYZ"),0) 
SET DECRYPT_VAR =   AESDECRYPT("My16CharacterKey", ENCRYPT_VAR,TEXTLEN(ENCRYPT_VAR),0)
