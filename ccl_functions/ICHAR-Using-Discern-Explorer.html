ICHAR( ) returns the ordinal value of an ASCII character. If the character expression that is passed to ICHAR() contains more than a single character, the ordinal value of the first character in the string is returned. ICHAR( ) is the converse of the CHAR() function.

Syntax

ICHAR(character_expr)

Metasymbol

character_expr 

A character expression.

Examples

The below example sets VAL equal to 65.

VAL = ICHAR("A")

The below example sets VAL equal to 66.

VAL = ICHAR("B")

The below example sets VAL equal to 67.

VAL = ICHAR("CDE")
