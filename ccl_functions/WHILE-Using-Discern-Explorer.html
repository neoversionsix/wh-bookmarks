The WHILE command repeats a process as long as some qualification is true. This is particularly useful for incrementing counters when setting up a loop (for example, when reading or writing to arrays or record structures).

Syntax

WHILE (qualification)
	while_item {,
	while_item}
ENDWHILE

Metasymbols

Qualification

A condition that evaluates to true or false. Qualifications can include IF, IN, and CASE commands and can be combined using the qualifiers AND, OR, NOT, and BETWEEN. Relational operators are used to define the condition. The syntax for qualification follows:

(qualification)
NOT qualification
qualification AND qualification
qualification OR qualification
expression relational_operator expression

while_item

A command or instruction performed by a WHILE statement. Valid WHILE items are command and report_item.

Command

A Discern Explorer instruction that can be executed by itself (for example, BROWSE, CREATE TABLE, DROP PROGRAM, and SELECT) or that can be included as part of another instruction (for example, CASE and WHILE).

report_item

A printable data element, calculation, or conditional used in a Reportwriter clause. The following are valid report items:

aggregate_expression

CALL TRACE

FOR command

BREAK command

CALL CENTER

IF command

CALL <uar>

CALL PRINT

MACRO command

CALL <sqlfunction>

CALL DEBUG

ROW and COL commands

CALL <subroutine>

DECLARE

select_expression assignment

CALL ECHO

CASE command

Text

CALL TEXT

variable

Examples

In the following example, the WHILE command is used in a Reportwriter section to print a name stored in the NAME_ARRAY. Two integer variables are used in the qualification. ELEMENT_NUM was incremented each time a name was placed in the array. COUNT is set to one outside the while loop and then incremented inside the while loop.

HEAD REPORT
     NAME_ARRAY[50] = FILLSTRING(45," ")	/*create the name array,
	                                  assumes we will never have
                                          more than 50 names in a nurse unit*/
HEAD NURSE_UNIT
     ELEMENT_NUM = 0	;reset element number for each nurse unit
     COUNT = 1	;reset count for each nurse unit
DETAIL
     ELEMENT_NUM = ELEMENT_NUM +1
     NAME_ARRAY[ELEMENT_NUM] = NAME	;write the name to the array
FOOT NURSE_UNIT
     COL 0 NURSE_UNIT
     WHILE (COUNT <= ELEMENT_NUM) 	;loop through the array
          COL 20 NAME_ARRAY[COUNT]	;print the name
          COUNT = COUNT+1
          ROW +1
     ENDWHILE
