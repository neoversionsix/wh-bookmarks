The CALL SCROLLINIT command tells Discern ExplorerÂ that a scroll window is used in the display. To change between multiple scroll windows, issue a CALL SCROLLINIT command to establish the current scroll window followed by CALL SCROLLTEXT to display text to the current scroll window. Use the CALL BOX command to display a box around the scroll window.

Note

The CALL SCROLLINIT command is only valid in the back end screen application programs that use the Paint command. This type of program was widely used in Cerner Classic, but in general, it is only used for utility programs in Millennium Platform.

Syntax

CALL SCROLLINIT (begin_row,begin_col,end_row,end_col)

Metasymbols

begin_col
The beginning column number (from 1 to 80 for 80-column mode and from 1 to 132 for 132-column mode) on the screen where the reverse-video box, scroll window, line, box, or text is displayed.

begin_row
The beginning row number (from 1 to 24) on the screen where the box, reverse-video box, text, line, or scroll window is displayed.

end_col
The ending column number (from 1 to 80 for 80-column mode and from 1 to 132 for 132-column mode) on the screen where the scroll window or box is displayed.

end_row
The ending row number (from 1 to 24) on the screen where the scroll window or box is displayed.

Examples

The following example creates a scroll window that starts at row 10, column 20, and ends at row 20, column 40:

CALL SCROLLINIT (10,20,20,40)
