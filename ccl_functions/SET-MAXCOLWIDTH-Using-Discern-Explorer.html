Use the SET MAXCOLWIDTH to change the maximum default length for display of a SELECT column that uses the SELECT INTO "MINE". The default is set to 2000.

Use GO at the end of the command to make the message option valid throughout a Discern Explorer session. If you do not use GO, the SET MAXCOLWIDTH command option is valid only during the execution of the program that called it and must be embedded in an executable program (between the CREATE PROGRAM and END commands).

Syntax

SET MAXCOLWIDTH = max_width [GO]

Metasymbol

max_width

The maximum default length for the displayer.

Examples

The following example sets the maximum width to 1000:

SET MAXCOLWIDTH = 1000 GO
