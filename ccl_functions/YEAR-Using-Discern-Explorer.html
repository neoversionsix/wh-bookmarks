YEAR allows you to return the integer year component of an expression that is a Julian date or rdb_date_time.

Syntax

YEAR(julian_date)
YEAR(rdb_date_time)

Metasymbols

julian_date

An expression that is anÂ Internal System Date.

rdb_date_time

A column that stores a datetime or function that returns a datetime.

Examples

In this example, if the date in the P.BIRTH_DT_TM column equals October 7, 1997, W1 is set to 1997. If the current date is December 24, 1998, W2 is set to 1998.

W1 = YEAR(P.BIRTH_DT_TM)
W2 = YEAR(CURDATE)

In this example, if the active_status_dt_tm is on 15-Aug-2023, active_year is set to 2023. If the current date is 15-Aug-2023, cur_year and fun_year are set to 2023. 2023_year is set to 2023.

select
	date = P.ACTIVE_STATUS_DT_TM,
	active_year = year(P.ACTIVE_STATUS_DT_TM),
	cur_year = year(curdate),
	fun_year = year(cnvtdatetime(curdate,curtime3)),
	2023_year = year(cnvtdate(08152023))
from
	person p
with maxrec = 10
