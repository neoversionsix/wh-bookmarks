The FILLSTRING() function defines a repeating string for a defined number of integer positions. For example, to create a line that is 60 characters long, instead of typing the underscore character (_) 60 times, you can use the FILLSTRING( ) function as shown in the below example. You then can call the expression any number of times in your query. This functionÂ is especially useful on the Report Layout screen when you want to create headings that are separated from the body of the text by a set of characters, such as a line.

Syntax

FILLSTRING(integer, literal_char)

Metasymbols

integer
An integer indicating the number of literal characters to return. If using a variable, the variable must be wrapped in the VALUE() function.

literal_char
A one-character literal. Quotation marks are required.

Examples

In this example, LINE is a 10-character string set to blank spaces, and LINE1 is a 60-character string set to "_".

LINE = FILLSTRING(10, " ")
LINE1 = FILLSTRING(60,"_")

This example uses the VALUE() function to allow using the variable LCNT as the first parameter.

declare lcnt = i4 with protect
declare str = vc with protect
for(lcnt = 1 to 10)
	set str = fillstring(value(lcnt),"a")
	call echo(str)
endfor
