EVEN( ) returns 1 if the integer expression specified is even or 2 if the integer expression is odd.

Syntax

EVEN(integer_expr)

Metasymbol

integer_expr

An expression that is an integer.

Examples

In the below example, because the variable V1 is set equal to ten (10), Discern Explorer displays the message: This is an even #.

SET V1 = 10
IF (EVEN(V1)=1)
   CALL ECHO("This is an even #.")
ELSE
   CALL ECHO("This is an odd #.")
ENDIF
