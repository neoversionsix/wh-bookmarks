MODIFY( ) returns 1 if the modify option is on, and returns 0 if the modify option is off.

Syntax

MODIFY(modify_option)

Metasymbols

modify_option

An option that can be set using the SET MODIFY command.

Examples

In the following example, the first call echo would most likely display 0 and the second call echo would most likely display 1:

call echo(modify("SPOOLFILE")) GO
SET MODIFY SPOOLFILE GO
call echo(modify("SPOOLFILE")) GO
