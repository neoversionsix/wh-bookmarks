By default, the PERCENT( ) function returns the value from the result set, where 50 percent of the values come before the returned value. The PERCENTILE( ) function can be used to specify a different percentile. It is used primarily within the PERCENT( ) function. If it is used in another context, PERCENTILE( ) returns the first parameter.

Syntax

PERCENTILE(expression, percent_num)

Metasymbols

expression
A numeric value.

percent_num
A numeric value representing the percentage.

Examples

Suppose the Res field on the Cust_Table contains the following values:

5

10

15

20

25

30

35

40

45

In the example below, PERCENT(C.RES) returns 25, PERCENT(PERCENTILE(C.RES,25)) returns 15, and PERCENT(PERCENTILE(C.RES,75)) returns 35.

SELECT
     C.RES
FROM CUST_TABLE C
ORDER BY C.RES
FOOT REPORT
	COL 10 "50%"
	COL 30 "25%"
	COL 50 "75%"
	ROW +1
	COL 10 PERCENT(C.RES)	";L"
	COL 30 PERCENT(PERCENTILE(C.RES,25))	";L"
	COL 50 PERCENT(PERCENTILE(C.RES,75))	";L"
	ROW +1
