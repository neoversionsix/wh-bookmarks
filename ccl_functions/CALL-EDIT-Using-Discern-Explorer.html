The CALL EDIT command calls the LSE, EDT, TPU, EMAC, or VI editor to create or modify a file. The user must specify the editor to be called in the CALL command.

Note

The CALL EDIT command is only valid in interactive Discern ExplorerÂ sessions.

Syntax

CALL EDIT (edit_file,editor)

Metasymbols

Edit_file

A string surrounded by quotes that contains the name of the file to create or modify for edit.

Editor

The name of the editor to be used. Valid editors are listed below.

ValueDescription

EDT

Use the EDT editor.

EMAC

Use the EMAC editor. (unix)

LSE

Use the LSE editor.

TPU

Use the TPU editor.

VI

Use the VI editor. (unix)

Examples

The following examples illustrate different ways to use CALL EDIT:

To edit a file named SCROLL.CCL in the default CCLUSERDIR: directory using the TPU editor, enter the following command:

CALL EDIT ("SCROLL.CCL", "TPU")

To use logicals and the vi editor, enter the following command:

CALL EDIT("CCLCUST:MY_PROGRAM.PRG", "VI")

To set a variable, to store the variable, and then issue the CALL EDIT command and pass that variable, enter the following command:

SET EDITOR = "EDT"
CALL EDIT("CCLUSERDIR:TEST_FILE.PRG",EDITOR)
