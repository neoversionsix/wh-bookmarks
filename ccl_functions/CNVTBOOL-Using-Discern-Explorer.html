CNVTBOOL () returns FALSE if the expression is 0 and TRUE if the expression is not 0. In Discern Explorer, TRUE and FALSE are system variables that are declared WITH CONSTANT() and initialized to 1 and 0 by the start up process. CNVTBOOL() can also be used to pass a Boolean type for a parameter to a plsql function.

Syntax

CNVTBOOL(expr)

Metasymbol

expr

An integer expression

Examples

In the following example, Temp_True is set to FALSE if Var is equal to 0; otherwise, Temp_True is set to TRUE:

SET TEMP_TRUE = CNVTBOOL(VAR)
