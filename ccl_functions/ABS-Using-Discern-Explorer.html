The ABS( ) function allows you to return the absolute value of a numeric expression.

Syntax

ABS(numeric_expr)

Metasymbol

numeric_expr
An expression that is an integer or real number.

Examples

In this example, if Y = -10, then X = 10.

X = ABS(Y)
