The CALL PRINTIMAGE command adds a PostScript snippet document into a dio=postscript report. You must use the NOFORMFEED control option with the CALL PRINTIMAGE command.

To use CALL PRINTIMAGE, you must have a PostScript snippet on the host environment and use PostScript printing.

Note

Bitmap images print faster when they have fewer colors. Because 24-bit color is not supported, you might need to change the color depth to 256 colors or grayscale, 16 colors, or monochrome. If you are using monochrome, images must have a width divisible by 32.

Complete the following steps to print a bitmap in a Discern Explorer report:

  1. Convert the PC bitmap to a PostScript file.

    Note

    When using Citrix to access Millennium Platform solutions, complete the below steps from the Citrix console.

    1. Open the DOS prompt and run the BMPFORMCNVT command from the WININTEL directory.

    2. Pass the bitmap file, new file name, and scale. The new file name and scale are optional. If the new file name is not used, the file name of the .BMP file is used by default. The new file name must contain at least one letter and cannot be comprised solely of numerical digits. If the scale is not used, a scale of 100 percent is assumed.

      "DOS>BMPFORMCNVT C:\MSOFFICE\CLIPART\HARDWARE_BMP"

      Note the scaled bitmap width and height.

    3. BMPFORMCNVT creates a .FRM and a .DCT file. Transfer these files to the HOST environment where the Discern Explorer program is located. When using FTP to transfer the files to the host, ensure that ASCII text mode is used. Do not transfer the files using binary mode.

    4. Open the DOS prompt and run FTP.

      DOS> FTP HOST1

    5. Sign in.

      FTP> PUT HARDWARE.FRM
FTP> PUT HARDWARE.DCT

      The files are copied from the default PC directory to the default directory on HOST1.

    6. Use the POS command to position the lower-left corner of the image in a Reportwriter clause. The Y coordinate (height) must be greater than or equal to the scaled bitmap height.
    7. Use the file and location from the HOST in the CALL PRINTIMAGE command.

Syntax

CALL PRINTIMAGE([file_name])

Metasymbol

file_name
The name of the file.

Examples

drop program image_test go
create program image_test

prompt "Enter Output Device: " = mine

select into $1
 d.seq
from dummyt d

head report
;;Loads image using the FRM
  call printimage("/user01/td5110/heartofcerner.frm")
  cnt = 0
  line = fillstring(50,"_")
  string = "TESTING STRINGS WITH VERTICAL LINE"

head page
;;This will print an image at the top of each page based on the
;;calcpos and call printimage command
  call print(calcpos(36,103))

  row +1
  call printimage("/user01/td5110/heartofcerner.dct")
  row +2

detail
  "{pos/288/144}{fr/0}" string
  row +1
  "{pos/144/144}{fr/1}" line
  row +1

with dio = 08,noformfeed,maxrow = 1

end
go
