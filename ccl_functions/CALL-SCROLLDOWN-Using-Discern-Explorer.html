The CALL SCROLLDOWN command controls the way a scroll window reacts when a user presses the DOWN arrow. CALL SCROLLDOWN determines whether Discern Explorer advances to the next line in the scroll box or moves the top line of the scroll window out of the box and moves the next available line not in the scroll window to the bottom of the box.

Note

The CALL SCROLLDOWN command is only valid in the back-end screen application programs that use the Paint command. This type of program was widely used in Cerner Classic, but in general, it is only used for utility programs in Millennium Platform.

Syntax

CALL SCROLLDOWN (previous_row,current_row,expression)

Metasymbols

current_row

The row that is selected as a result of the user pressing the DOWN arrow.

Expression

An expression is the highest-level metasymbol among constants, field names, variables, and so on. The following expressions are valid:

alias.field_name
array_name [dimension]
array_name [dimension1, dimension2]
array_name [dimension1, dimension2, dimension3]

Constant
expression arithmetic_operator expression
-expression
(expression)
function(expression{,expression})
parameter
record_name{->field_name}
table_name.field_name
variable

Note

Square brackets are part of the array_name syntax; they do not indicate an optional part of the syntax.

previous_row

The last row that was selected before the user pressed the DOWN arrow.

Examples

The example below might be part of a CASE statement or a complex IF statement that checks for the value of CURSCROLL. CURSCROLL is a variable used by Discern Explorer to keep track of which keys are pressed when a scroll box is displayed. See the SELECT command and the CALL ACCEPT command for more information on CURSCROLL.

IF (CNT < MAXCNT)
SET CNT = CNT + 1
	IF (AROW = NUMSROW)
	CALL SCROLLDOWN(NUMSROW,NUMSROW,BUFFER[CNT])	;scroll bottom of window
	ELSE
	SET AROW = AROW + 1
CALL SCROLLDOWN(AROW-1,AROW,BUFFER[CNT])	;scroll above bottom
ENDIF
ENDIF

The following example performs several actions:

  • Advances to a line in the box, usually the next line (the first command).
  • Advances the first line of information in the box outside the box, and moves all lines below that first line up one line. It then displays the next line previously not shown in the box in the last line of the box (the second command).
CALL SCROLLDOWN(AROW-1,AROW,BUFFER[CNT])
CALL SCROLLDOWN(NUMSROW,NUMSROW,BUFFER[CNT])

Several variables have been defined for this example in earlier parts of the program.

CNT is the current array occurrence being displayed in the scroll box.

MAXCNT is the total number of occurrences in the array.

AROW is the current row number in the scroll box.

NUMSROW is the maximum number of rows in the window.

BUFFER is an array that was completed by selecting information from a table in the Discern Explorer Data Dictionary using the SELECT command earlier in the executable program.
