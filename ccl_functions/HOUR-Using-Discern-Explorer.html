Returns the integer hour (HH) of an expression that is in (HHMM) format or a rdb_dt_tm column. An integer from 0 to 23 is returned.

Syntax

HOUR(time)
HOUR(rdb_dt_tm)

Metasymbols

time
An expression that is in HHMM format.

rdb_dt_tm
A column that stores the date and time.

Examples

In this example, if the current time is 14:30 (2:30 p.m.), H1 is set to 14.

H1 = HOUR(CURTIME)

In this example, if the ARRIVE_DT_TM column is 01-JAN-1999 11:30:39.00, H2 is 11.

H2 = HOUR(E.ARRIVE_DT_TM)
