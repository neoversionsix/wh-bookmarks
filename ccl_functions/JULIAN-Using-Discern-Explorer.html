JULIAN( ) returns the Julian date or the day of the year for the specified date.

Syntax

JULIAN(julian_date)
JULIAN(rdb_date_time)

Metasymbols

julian_date

An expression that is an Internal System Date.

rdb_date_time

A column that stores a datetime or function that returns a datetime.

Examples

When 01012014 is entered at the prompt, the example below sets DAY to 1. When 12312014 is entered at the prompt, the following example sets Day to 365:

drop program julian_test go
create program julian_test

Prompt "Enter Calender date in MMDDYYYY format" 

select into noforms
	day = julian(cnvtdate($1))
from
	dummyt
with format, separator = " "
end
go

In the following example, if the Person record was updated on 01-FEB-2014, Day is set to 32.  If the current date is 15-Aug-2023, cur_julian and fun_julian are set to 227. Day_julian is set to 227.

select 
    day = julian(p.updt_dt_tm),
	cur_julian = julian(curdate),
	fun_julian = julian(cnvtdatetime(curdate,curtime3)),
	day_julian = julian(cnvtdate(08152023)) 
from person p
where p.person_id = 12345.0
with format, separator = " "
