The DECODE( ) function is used with a PLAN clause that uses ORJOIN or a SELECT that uses the DONTCARE or OUTERJOIN control options. DECODE( ) can be used to determine which side of the ORJOIN a row in the result set is from or if the row in the result set contains values from the DONTCARE or OUTERJOIN tables. DECODE( ) returns the first result following a table_name.column_name value that is not a placeholder in the result set. The last result is returned if all the expressions from the first to the nth are placeholders. When Discern Explorer creates the result set, placeholders are used for columns that are from the other side of the ORJOIN or from tables that are a part of the DONTCARE or OUTERJOIN that do not have a qualifying record.

Syntax

DECODE(table_name.column_name, result{,table_name.column_name, result}[,result])

Metasymbols

column_name

The name of the column from the Discern Explorer Data Dictionary to be updated, inserted, or read.
result

An expression that can be returned from the DECODE( ) function.
table_name

The name of the table from the Discern Explorer Data Dictionary to be updated, deleted, or read.

Examples

In the following example, VAL is set to 1 if table A contains a non-placeholder value in the pseudo column SEQ. If A.SEQ is a placeholder, and B.SEQ is not a placeholder, VAL is set to 2.If A.SEQ and B.SEQ are placeholders and C.SEQ is not a placeholder, VAL is set to 3.If A.SEQ, B.SEQ, and C.SEQ are all placeholders, VAL is set to 0.

VAL is set equal to the value following the first table (reading from left to right) that has a non-placeholder value in the SEQ column. If none of the tables have a non-placeholder value in the SEQ column, VAL is set to 0.

VAL = DECODE(A.SEQ,1,B.SEQ,2,C.SEQ,3,0)

In the following example, CHECK is set to C if the row in the result set is from the Order_Comment side of the ORJOIN. CHECK is set to R if the row is from the Result side of the ORJOIN.

SELECT
NAME = SUBSTRING(1,30,P.NAME_FULL_FORMATTED),
ORDER_MNEMONIC = SUBSTRING(1,30,O.ORDER_MNEMONIC),
CHECK = DECODE(OC.SEQ,"C",R.SEQ,"R"),
R.RESULT_ID,
OC.LONG_TEXT_ID

FROM PERSON P,
ORDERS O,
ORDER_COMMENT OC,
RESULT R,
DUMMYT D1,
DUMMYT D2

PLAN P WHERE P.PERSON_ID >0

JOIN O WHERE O.PERSON_ID = P.PERSON_ID
AND O.ORDER_ID >0

JOIN D1
JOIN (OC WHERE OC.ORDER_ID = O.ORDER_ID)

ORJOIN D2
JOIN (R WHERE R.ORDER_ID = O.ORDER_ID)
