REPORTINFO( ) allows dynamic help scripts to get access to select target list heading and result buffer.

Syntax

REPORTINFO(option)

Metasymbol

option
A number representing the information to be returned.

Option

Return

0

This option returns the length of the buffer.

1

This option returns the heading.

2

This option returns the target buffer.
