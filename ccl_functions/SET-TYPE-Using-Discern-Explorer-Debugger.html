The SET TYPE command determines the increment by which the debugger scrolls through the lines of the source when you issue the TYPE command. The default is the number of source lines displayed in the Debugger window.

Syntax

S[ET] T[YPE] increment_num

Metasymbol

increment_num

The number of lines that the debugger scrolls through the source.

Example

The following example sets the type increment to ten. The next time you issue the TYPE command, it scrolls through the source ten lines at a time.

S T 10
