The SET command assigns a value to a variable, record structure item, or array item. If the record structure item or array item does not exist and error or warning is generated. If the variable does not exist, it is created. The SET MODIFY PreDeclare command is used to generate a warning message if the variable was not previously created using the DECLARE command. If SET MODIFY PreDeclare is active, the SET command will create the variable and generate a warning. If SET TRACE Deprecated E and SET MODIFY PreDeclare are active, an error is generated and the variable is not created..

GO is used to execute the SET command outside of a CREATE PROGRAM and END command.

The WITH PERSIST option can be used in a child process to make a variable known when control returns to a parent process. However using this option is not recommended. Generally the variable should be created in the parent process and used in the child process. In rare cases where a child process needs to create a variable and make it known to the parent process, the DECLARE command specifying the PERSISTSCRIPT scope should be used in the child process instead of using the WITH PERSIST option in the SET command.

Syntax

SET variable = expression [WITH PERSIST] [GO]

Metasymbols

variable
The variable name, record structure item, or array item to set to a value.

expression
An expression is the highest-level metasymbol among constants, field names, variables, and so on. The following expressions are valid:

alias.field_name
array_name [dimension]
array_name [dimension1, dimension2]
array_name [dimension1, dimension2, dimension3]
constant
expression arithmetic_operator expression
-expression
(expression)
function(expression{,expression})
parameter
record_name{->field_name}
table_name.field_name
variable

Note

Square brackets are part of the array_name syntax; they do not indicate an optional part of the syntax in this instance.

Examples

The following example sets a variable named CNTR to 10. The GO indicates that the SET command is used outside of a CREATE PROGRAM END command.

SET CNTR = 10 GO

The following example increments the CNTR variable by one.

SET CNTR = CNTR +1

The following example declares a two position array named c3_array. Each position of the array can hold up to three characters. The first position of the array is set to "One" and the second position of the array is set to "Two".

declare c3_array[2] = C3

set c3_array[1] = "One"
set c3_array[2] = "Two"
