Removes a file from the specified directory, the default is CCLUSERDIR. The function returns 1 if the file is removed successfully and 0 if an error is encountered.

Syntax

REMOVE(file_name)

Metasymbol

file_name
The name of a file. The default location is CCLUSERDIR unless otherwise specified.

Examples

The following example incorporates the REMOVE( ) function and deletes the test.dat file from the CCLUSERDIR: directory.

STAT = REMOVE("test.dat") go

The following example deletes test_report.prg from the cust_script directory:

SET STAT = REMOVE("cust_script:test_report.prg") GO

Note

Call DCL can be used to delete multiple files. Use extreme caution when using wildcard characters to delete multiple files. The following example deletes all files in CCLUSERDIR with names that begin with my_test_file and end with .prg

SET SYSCMD = "rm $CCLUSERDIR/my_test_file*.prg" go
SET LEN = SIZE(TRIM(SYSCMD)) go
SET STATUS = 0 go
CALL DCL(SYSCMD,LEN,STATUS) go
