RENAMEREC() renames a record structure (defined with the RECORD command). It was introduced in version 8.7.1 of Discern Explorer. RENAMEREC returns 1 for success and 0 for error.

Syntax

RENAMEREC(rec_name_from, rec_name_to)

Metasymbols

rec_name_from
The name of a record structure that was previously defined using the RECORD command. The record structure must already exist.

rec_name_to
The new name to assign to the record structure. The record structure with this name must not already exist.

Examples

The following command renames Arec to Brec:

	SET STAT = RENAMEREC(AREC,BREC)
