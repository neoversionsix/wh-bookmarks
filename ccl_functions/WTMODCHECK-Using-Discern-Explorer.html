WTMODCHECK returns a weighted modulus 10 check digit.

WTMODCHECK ( ) is used extensively in barcode labels and error detection. If a bar code gets printed incorrectly, the incorrect drug could be administered.

Error Detection

An error-detecting code is a way of transmitting data so that most common mistakes are detected at once, before they can cause any damage. The code performs error detection, but not error correction.

Generally, the number to be transmitted is followed by a number of check digits - most often one for simple error detection. Then when the number is transmitted, another calculation can be done at the receiving end to check that the received number (including the check digit) is valid.

Examples

In the example, WTMODCHECK( ) returns the following weighted modulus 10 check digit: 3*4 + 1*3 + 3*8 + 1*2 + 3*7 = 10 - (62 mod 10) = 8. The final printed result is "438278".

X = "43827"
CALL PRINT(BUILD(X,WTMODCHECK(10,X)))
