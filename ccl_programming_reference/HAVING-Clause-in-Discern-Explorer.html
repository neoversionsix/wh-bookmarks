The HAVING clause is used in combination with the GROUP BY clause. Use the HAVING clause to restrict the rows from the GROUP BY clause that are returned. First, any qualification in the WHERE clause will restrict the detail rows. Second, the groups are formed from the GROUP BY clause. Finally, the HAVING clause only displays those groups that satisfy the condition.

The following query returns a count of the encounters in each nursing unit where the discharge datetime is NULL.

SELECT	E.LOC_NURSE_UNIT_CD,
	COUNT(*)
FROM	ENCOUNTER E
WHERE	E.DISCH_DT_TM = NULL
GROUP BY	E.LOC_NURSE_UNIT_CD
GO

The following query uses the HAVING clause to only returns the nursing units where the count of the encounters is greater than 10.

SELECT	E.LOC_NURSE_UNIT_CD,
	COUNT(*)
FROM	ENCOUNTER E
WHERE	E.DISCH_DT_TM = NULL
GROUP BY	E.LOC_NURSE_UNIT_CD
HAVING 	COUNT(*) > 10
GO
