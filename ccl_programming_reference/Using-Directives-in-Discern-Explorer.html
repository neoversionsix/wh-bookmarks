Discern Explorer directives are single-line commands that instruct Discern Explorer to perform a task other than a query. For example, the BANNER directive redisplays the Discern Explorer banner at the top of the screen. When you use directives, follow the rules listed below.

  • Each directive (except the DIR and OS_Command directives) must be preceded by a percent sign (%).
  • The DIR and OS_Command directives must be preceded by a dollar sign ($).
  • All directives must begin in the first column.
  • Enter the entire directive name, or just the first letter (such as %J).

Valid directives are described briefly in the table below. See the Command Reference Help for additional information and examples. Descriptions of some of these directives are included in this section where appropriate.

Directive

Description

%BANNER (%B)

Redisplays the Discern Explorer banner at the top of the screen.

%CLEAR (%C)

Turns the logging of Discern Explorer commands to the Discern Explorer history file on and off.

%#DEFINE

Creates a conditional macro variable. Conditional macro variables are used to apply conditions at compile time.

%DISPLAY (%D)

Turns the display of input commands to the user on or off.

%EDIT (%E)

Edits any text file.

%#ELSE

Begins the else portion of a conditional block based on the existence or non-existence of a conditional macro variable at compile time.

%#ENDIF

Ends the IFDEF or IFNDEF conditional compile block.

%HELP (%H)

Displays descriptive text for the specified topic.

%#IFDEF

Begins a conditional block based on the existence of a conditional macro variable at compile time.

%#IFNDEF

Begins a conditional block based on the non-existence of a conditional macro variable at compile time.

%INCLUDE (%I)

Reads a Discern Explorer query stored in a file and executes its commands.

%JOURNAL (%J)

Edit the Journal file and include the contents when you exit the editor.

%KEDIT (%K)

Edits the most current Run-Time Library (RTL) log file from a session.

%LEDIT (%L)

Changes the editor currently being used to the editor specified.

%OUTPUT (%O)

Redirects all output to a specified ASCII file until the session ends or until the %O directive is issued without a specified file name.

%PRINT (%P)

Prints a message at compile time.

%RUN (%R)

Combines the %EDIT and %INCLUDE directives in one directive.

%SHOW (%S)

Displays the current values of Discern Explorer ranges, defines, select options, and directive settings.

%TERMINAL (%T)

Sets the terminal to character or line mode.

%#UNDEF

Deletes a conditional macro variable at compile time.

%VERSION (%V)

Specifies a version number to stamp on the executable program.

%Z

Displays the contents of the recall buffer.

$DIR

Displays files that match a file specification.

$OS_Command

Branches to an operating system subprocess and executes an operating system command.
