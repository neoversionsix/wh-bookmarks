The Reportwriter section of a Discern Explorer query consists of the seven clauses listed below, each of which defines a specific aspect of the report format.

  • HEAD REPORT
  • HEAD PAGE
  • HEAD group expression
  • DETAIL
  • FOOT group expression
  • FOOT PAGE
  • FOOT REPORT

Each of these clauses is described separately in the sections above.
