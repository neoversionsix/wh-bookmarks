The FROM clause in the SELECT command identifies the tables from which Discern Explorer extracts information. The following syntax is from the FROM clause:

FROM
	table_name [alias],
	table_name [alias]

When selecting information from more than one table, list all the tables in the FROM clause separated by commas. Do not place a comma after the last table listed. For example:

FROM	PERSON,
	ORDERS,
	ORDER_CATALOG
