Each Discern Explorer file has a unique filename. File names follow the convention file_name, period, file_extension, for example, PATNAMES.CCL. On OpenVMS systems, the file_extension is followed by a semicolon, and version_number. For example, PATNAMES.CCL;1 filenames are associated with specific types of files, including files you create (such as scripts and programs) and files created by Discern Explorer (such as log files). For scripts and programs that you create, you define the filename. Although it is not necessary to include the extension when you define the filename, you can use the same extension with all of your programs. This makes it easier to identify at the operating system directory level.
