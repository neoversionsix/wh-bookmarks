When you create variable expressions, initialize those expressions to some value in the HEAD REPORT clause. Initializing a variable determines its data type and display length. Creating a variable as an integer defaults that variable to a display length of 11 bytes. A real number defaults to 14 bytes for display purposes. A string variable defaults to the number of characters in the initial value as the display width. The following example creates three variables in the HEAD REPORT clause. COUNT is an integer data type, RESULT is a real data type, and NAME is a character data type. The example then shows how values could be assigned to the variables in the DETAIL section.

HEAD REPORT
	COUNT = 0
	RESULT = 0.0
	NAME = FILLSTRING(30, " ")
DETAIL
	COUNT = COUNT +1
	RESULT =CNVTREAL(COUNT) / 25
	NAME = SUBSTRING(1,30,P.NAME_FULL_FORMATTED)
