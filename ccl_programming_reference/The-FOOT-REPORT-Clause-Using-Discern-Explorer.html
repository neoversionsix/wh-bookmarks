The FOOT REPORT clause is used to perform report items at the end of a report, like printing a summary page.

Keep these points in mind when using the FOOT REPORT clause:

  • The FOOT REPORT clause is optional.
  • If a FOOT REPORT clause is used a HEAD REPORT clause must also be used.
  • A SELECT command can include only one FOOT REPORT clause.
  • A FOOT REPORT clause must include at least one report item.

Syntax

[FOOT REPORT
	report_item {,
	report_item }]

Examples

The most common use of the FOOT REPORT clause is to print grand totals, averages, maximums, minimums, and to perform other report items as necessary. The following example uses a FOOT REPORT clause to print grand totals and a line of text to note the end of the report.

FOOT REPORT
	ROW +5
	CALL CENTER("*** Grand Totals For Report ***",0,120)
	ROW +1
	COL 62 "Total number of days: "
	COL 95 SUM(LOS)
	ROW +1
	COL 62 "Total number of patients: "
	COL 95 COUNT(NAME)
	ROW +1
	COL 62 "Total Patients with LOS > 5: "
	COL 95 COUNT(NAME WHERE LOS >5)
	ROW +1
	COL 62 "Average length of stay: "
	COL 95 AVG(LOS)
	ROW +5
	CALL CENTER("*** END OF REPORT ***",0,120)
