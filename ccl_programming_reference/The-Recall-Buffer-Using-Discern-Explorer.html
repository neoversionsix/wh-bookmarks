Discern Explorer uses a recall buffer of ten lines to temporarily store the commands entered on the command line. To redisplay your previous ten entries individually, press the UP arrow. If you press the UP arrow once, Discern Explorer displays the previous line you entered. If you press the UP arrow 11 times, Discern Explorer displays the last ten lines you entered and returns to the most current line entered.

Discern Explorer discards duplicate consecutive lines from the recall buffer. Therefore, if you enter P.NAME_LAST_KEY, on three successive command lines, only the first entry is retained in the recall buffer even though all three entries are used in the Discern Explorer query.

To display all the lines stored in the recall buffer, press SELECT. (The SELECT key is a key on a standard VT keyboard. Check your terminal emulation key mapping to determine what key on your PC is mapped to the SELECT key. Most often the End key is used.)
