Reportwriter syntax uses the report_item metasymbol. A report item is a printable data element, calculation, or conditional used within a Reportwriter clause. The underlying goal of any report item is the generation of formatted results, although not all report items are actually displayed. For example, the statement ROW +1 is a report item that advances the printing to the next line. An IF() or WHILE() statement used in a Reportwriter section is also a report item that acts as a conditional. The report_item metasymbol represents the following Discern Explorer elements:

  • Aggregate Function
  • CALL Commands
  • CASE Command
  • EXPRESSIONS
  • FOR() Loop
  • IF() Command
  • MACRO Command
  • ROW or COL Commands
  • SELECT Expressions
  • TEXT
  • WHILE( ) Loop

When you use a Reportwriter section, you must tell Discern Explorer which report items are used, when they are used, where they are used, and how to perform them. The use of each of the above report items is explained separately in the following sections.
