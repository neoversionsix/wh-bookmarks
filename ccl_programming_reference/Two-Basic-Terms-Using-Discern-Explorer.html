Two terms that are critical to understanding the process of creating an executable program are source code and object:

  • Source code refers to the commands that you enter that follow the rules and syntax required by Discern Explorer. The file that you create using an editor is considered the source code.


  • Discern Explorer takes the source code and creates an object, which is a series of machine language instructions that is in a format the computer can understand. The object or executable program is stored in the object library.
