The process of running a program can be broken down into the following three distinct steps:

  1. The source code is interpreted and checked for errors in syntax.
  2. The program is compiled. Compiling is the process in which your source code (Discern Explorer commands) is converted to binary code (the object) so the computer can read your instructions more efficiently.
  3. The program is executed (the binary code is processed step by step until your program is completed).
