The HEAD REPORT clause can also be used to define a title to be printed at the top of the report. The following example defines a title for a report using the HEAD REPORT clause:

HEAD REPORT
	LINE_D = FILLSTRING(90,"=")
	LINE_S = FILLSTRING(90,"-")
	CALL CENTER("*** CERNER'S INPATIENT REPORT ***",0,89)
	COL 0 CURDATE "MM/DD/YY;;D"
	COL 75 CURTIME "HH:MM;;M"
	ROW+1 LINE_D
	ROW+2

The example uses the CALL CENTER statement to print a report title. The first item to print on the page is the text string *** CERNER'S INPATIENT REPORT ***. The CALL CENTER command centers the string between Columns 0 and 89. On the same line, the current date is printed to the left, starting at Column 0. The current time is printed starting at Column 75.

The reserved variables CURDATE and CURTIME are used to display the current date and time. Each column was formatted using an appropriate display option so the date and time are more readable.

The system then advances one line (ROW +1), and the LINE_D variable is printed. This variable is initialized in the HEAD REPORT clause using the FILLSTRING( ) function. LINE_D is equal to 90 equal sign (=) characters. The last item in the HEAD REPORT clause (ROW +2) advances two lines to create a blank line.
