Standard column types have been defined and used throughout the Millennium Platform data model. Columns have been named using a standard naming convention to reflect the standard column types. Understanding these column types help you with writing Discern Explorer queries. Standard column types are:

  • Internal Identifier ( _ID columns )
  • Index Character ( _KEY columns )
  • Indicator ( _IND columns )
  • Date and Time ( _DT_TM columns )
  • Codes ( _CD columns )
