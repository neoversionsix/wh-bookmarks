You can write Discern Explorer programs at the command line or with the Visual Explorer or Discern Visual Developer application. Once a program is written, you can execute the program without actually accessing Discern Explorer command line, Discern Visual Developer, or Visual Explorer. Use any of following solutions to execute the program:

  • Explorer Menu (ExplorerMenu.exe)
  • System Operations (OPSVIEW Scheduler.exe)
  • Other Cerner Applications
  • Operating System
  • MPages

Release Considerations

Oracle no longer supports Explorer Menu. For more information, see FLASH16-0165-15. This solution is replaced by Discern Analytics 2.0. Contact your Client Results Executive (CRE) for information on implementing Discern Analytics 2.0.
