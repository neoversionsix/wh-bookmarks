Discern Output Viewer is a component used in many Millennium Platform applications to display Discern Explorer (CCL) reports. The report output can be any of the currently supported output types: ASCII, RTF, PDF, PS, CSV, and HTML.

Discern Visual Developer (DiscernVisualDeveloper.exe) is a tool for use in developing or modifying Discern Explorer (CCL) programs, and its graphical tool Layout Builder can be used to create a new report quickly. For help with developing new Discern Explorer (CCL) programs using Discern Layout Builder, see the Discern Explorer Layout Builder Tutorial.

ASCII, RTF, PDF, PS, and CSV are report output types used to display static content, whereas HTML output can be used to display interactive content, such as HTML. Additionally, Discern Output Viewer provides a framework that an HMTL page can use to communicate with Millennium Platform through Discern Explorer (CCL) programs or interact with PowerChart or DiscernNotify applications. For example, when a link in an HTML report is clicked, it can execute a Discern Explorer program; open a Millennium Platform application without requiring the user to log in again; open a patient chart if a patient ID is specified; or if it is a message in DiscernNotify, it could send an event to delete the message. This provides a powerful framework that you can use to quickly create a new solution to compliment or enhance your workflows.

This topic shows the available API provided by the framework that can be used in an HTML page. CCLLINK, APPLINK, CCLEVENT, CCLNEWSESSIONWINDOW, CCLLINKPOPUP, MPAGES_EVENT, and MPAGES_SVC_EVENT are special Javascript functions that can be used.

For examples displaying HTML content and Javascript functions, see Building a Discern Explorer (CCL) Program to Display HTML Content.

Note

For use of multiple functions, separate the content names by commas without spaces. For example: <META content='CCLLINK,APPLINK,CCLNEWSESSIONWINDOW' name='discern'>
