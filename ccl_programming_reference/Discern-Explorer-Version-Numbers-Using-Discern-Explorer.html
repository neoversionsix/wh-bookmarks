The version-numbering scheme for Discern Explorer differs from that of the various clinical information systems available from Oracle.

Discern Explorer is designed to work with all versions of Oracle systems. To represent Discern Explorer as being tied to one of those versions is inaccurate; therefore, a version-numbering unique to Discern Explorer has been employed. The Discern Explorer version is represented using a major, minor, and sub-minor version number. The Reserved Variables CURREV, CURREVMINOR, and CURREVMINOR2 are equal to the major, minor, and sub-minor version numbers. Before Discern Explorer version 8.10.0, the Reserved Variable CURCCLREV is equal to the complete version number. In Discern Explorer versions 8.10.0 and higher, the Reserved Variable CURCCLVER is equal to the complete version number. Discern Explorer versions 9.0.0 and higher are used in 64-bit platforms. Discern Explorer versions earlier than 9.0.0 are used in 32-bit platforms.
