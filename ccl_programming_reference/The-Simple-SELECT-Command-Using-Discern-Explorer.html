The SELECT command is the most important command in Discern Explorer. The SELECT command enables you to do the following actions:

  • Perform simple queries.
  • Flex a program based on specific conditions.
  • Create detailed reports with user-defined headers, footers, and summary pages when combined with Reportwriter.
  • Define programs that give the user options on how the information is processed.

The SELECT command has its own syntax. This section describes the simple SELECT command, along with related clauses.

Although many examples of how the SELECT command can be used to extract data from system files are included in this section, these examples do not represent all uses for the SELECT command.

Note

All examples in this section were created by entering the commands on the Discern Explorer command line and then entering GO. See The Command Line in the Introduction section, for additional information about the command line.

You also can use the editor to create and execute the commands in a file using the %EDIT, %INCLUDE, or %RUN directives. Longer or more complex SELECT commands should be entered using an editor.

In a simple query, the SELECT command can perform the following tasks:

  • Identify the information that will be extracted from the database, and the tables from which the information will be extracted.
  • Format the information to be displayed.
  • Establish conditions that must be met before the information can be included in the output.
  • Organize the output.
  • Define options that control the entire query (such as setting the maximum number of columns or rows to be included in the display, or identifying a character to be displayed between each column).
  • Omit duplicate information from the output.
  • Direct the output of the query to a printer, a file, or the screen.

The syntax for the simple SELECT command is shown below.

SELECT
[DISTINCT]
[INTO
	output_device]
	select_expression["template;display_qualifier;display_format"] {,	select_expression ["template;display_qualifier;display_format"] }
[FROM
	table_name [alias] {,	table_name [alias] }]
[WHERE
	qualification]
[GROUP BY
	expression]
[HAVING
	Condition]

[ORDER
	expression [DESC] {,	expression [DESC] }]
[WITH
	select_option {,	select_option }]
GO
