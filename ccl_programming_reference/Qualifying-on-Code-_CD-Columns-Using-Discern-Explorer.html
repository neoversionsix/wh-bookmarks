It is more efficient to store numbers than it is to store text. Because of this, text strings that are repeated in multiple records are assigned a numeric value, or code. Code_values are used extensively throughout the Millennium Platform environment and can be identified by the _CD ending of a field name. For example, the SEX_CD field is found on the PERSON table. When selected, numeric values are displayed instead of the person's sex. The system assigns the code values. Code values vary from site to site and environment to environment. For example, the code value for female maybe 196 at one site and 15075 at another. Typically three code_values exist for SEX_CD - one each for male, female, and unknown.

To find out what code_values are assigned, you need to look at the code set for the field. See Determining Code_Sets using Discern Explorer to learn how to determine the code set for a field.

The codes that are assigned to text are located on the CODE_VALUE table.

See Qualifying on CODE_VALUE Fields using Discern Explorer to learn how to qualify on code fields.
