MPAGES_SVC_EVENT is a routine used to launch into the Discern MPages web service.

Note

This event type is only valid with an MPages license.

For this routine to work, the following META tag must be included in the HEAD of the html file: <META content='MPAGES_SVC_EVENT' name='discern'>

Syntax

MPAGES_SVC_EVENT(uri,params)

Metasymbols

uri

An address specifying the report to be executed.

params

The parameters for the report.

Examples

The following example runs the report ARH_RECENT_LOGINS using the Discern MPages web service:

<a href='javascript: MPAGES_SVC_ EVENT("http://ipvmdiscernmp:8080/discern/mpages/reports/arh_recent_logins","MINE,1")'>
