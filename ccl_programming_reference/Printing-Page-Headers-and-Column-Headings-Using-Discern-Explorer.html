The example below places the page number and column headings at the top of each page. Absolute column references are used to print the column headings. The same column references would be used in the DETAIL clause to print the data elements. The FILLSTRING( ) function is used to create a line under the column headings by adding an expression called LINE_S to the HEAD REPORT clause.

HEAD REPORT
	 LINE_S = FILLSTRING(90,"-")
HEAD PAGE
	;print the page header
	COL 0 "PAGE: "
	COL 7 CURPAGE "###"
	ROW +1
	;print the column headings
	COL 0 "Name:"
	COL 50 "Sex:"
	COL 60 "Birth Date:"
	COL 75 "Room-Bed:"
	ROW +1 LINE_S
	ROW +1
