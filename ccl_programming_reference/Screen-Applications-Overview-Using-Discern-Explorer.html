A screen application is an interactive program that uses commands to control the placement of data on the screen. It is important to understand that these screen applications can only be executed from command-line Discern Explorer using a terminal emulator.

Discern Explorer offers similar functionality using Visual Explorer or Discern Visual Developer to create the programs and the Explorer Menu to execute the programs. Functionality exists in Visual Explorer and Discern Visual Developer to create Prompt Forms that allows similar functionality to Screen Applications.

The phrase Screen Application is used to specify programs that use the PAINT clause the CALL BOX, CALL LINE, and CALL TEXT commands to provide information to the user during the execution of the program. Other features and commands referenced in this chapter (such as the use of Labels, GO TO statements, EXECUTE commands and other programming constructs) can be used without using the PAINT clause.

Screen applications have four distinct parts:

CALL commands
SET commands
SQL commands
Programming Constructs
