When using device-independent output (DIO) commands in your Reportwriter program, the general syntax rules listed below apply, regardless of the commands you are using.

The SELECT that creates the formatted output must define the correct output device using the syntax, SELECT INTO output_device. For example, SELECT INTO $OUTDEV. To select a DIO report into memory, you have to use SELECT INTO "CURDIO".

Add a DIO control option in the WITH clause to define the DIO device, such as WITH DIO = POSTSCRIPT. You can specify the device by using the name or the number of the device in your program. Valid DIO control options are listed under the Control Options topic. You also can review these options by entering "%HELP" (%H) at the Discern Explorer command line.

DIO commands must be enclosed in braces ( { } ) and must be enclosed in quotation marks in Reportwriter statements. For example, COL 0 "{COLOR/20}{B}".
