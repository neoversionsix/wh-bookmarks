The DIO COLOR command is implemented on PostScript devices as black, white, and shades of gray. Contrasting variations are shown in these examples, indicating some of the print highlighting that can be achieved.

Colors 0—10

Changes to the color of text are controlled with color values zero (black) through ten (white).

Note

The example of color 10 shows white characters on a white background.

Colors 11—20

The colors shown below are used to create one color of background and a contrasting color for the text automatically. Color 11 gives a black background and color 20 gives a very light gray background. For colors 11-15, the text is white. For colors 16-20, the text is black.

Colors 21—31

The colors shown below are used to create a shaded box automatically. After a COLOR command is issued in the range of 21—30, any subsequent BOX commands will create a shaded area for the size of the box. The color of the text inside the box is determined by the last issued color in the range of 0—10. For purposes of this example, color 10 (white) is used for the text color on box colors 21—25, and color 0 (black) is used for the text color on box colors 26—31. Note that color 31 gives a true box. Color 31 is the default for the box if a COLOR command for colors 21—30 is not issued.
