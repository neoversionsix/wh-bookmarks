The %H directive displays a Help menu containing documented Discern Explorer features. Use the ARROW keys to select a menu item or enter the number of a menu item and press ENTER to display helpful information on the topic. The Help is displayed in the Discern Explorer displayer. See The Displayer Using Discern Explorer for an explanation of how to use the displayer. 

Complete the following steps to open the Help:

  1. Enter %H to active the Help.
  2. Select COMMANDS from the menu. The displayer opens a list of Discern Explorer commands.
  3. Press F to open Find Mode.
  4. Enter Topic-Commands-Select and press ENTER on the numeric keypad.
  5. Press the DOWN ARROW to see the Help information for the SELECT command.
  6. Press ENTER three times to exit the displayer and the Help menu.

See Command Reference Help in Discern Explorer for the complete syntax of the Discern Explorer language.
