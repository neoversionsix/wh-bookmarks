Record structures allow you to temporarily store data in memory. Record structures are predominately used by OracleÂ middleware to pass information between a front-end application and a back-end script program. The front end creates a request record structure that is sent through the middleware to execute a specific Discern Explorer program. That program creates and populates a reply record structure that is sent back through the middleware to the front-end application. Record structures also are used by Discern Explorer report programs to temporarily store data in memory.
