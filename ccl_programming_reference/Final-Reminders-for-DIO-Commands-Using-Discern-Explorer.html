Keep the following points in mind when creating Reportwriter programs that use DIO commands:

  • The POSITION command gives you more control over the placement of data.
  • DIO commands are distinguished by braces ( { } ). The syntax for each command is in the DIO COMMANDS section.
  • When using DIO commands, Discern Explorer uses a buffer of 300 columns. Increase the lines in the buffer by using the statement ROW+1.
  • The DIO commands are stored in the buffer as text. Therefore, you must treat the commands like text when using them in the Reportwriter syntax by enclosing them in quotation marks. The printer executes the commands when the report is printed.
  • Use SELECT INTO MINE to determine whether the DIO commands are being stored in the buffer correctly.
