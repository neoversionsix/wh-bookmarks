CCLLINK is the routine used to build a link to another Discern Explorer (CCL) program in an HTML document. By clicking the link, the program runs with the provided prompt values and displays in the Discern Output Viewer (DiscernOutputViewer.ocx). This is useful for building report menus or linking one report to another report.

Syntax

CCLLINK( reportname,prompts,linkdestination)

Note

For this routine to work, the following META tag must be included in the HEAD of the html file: <META content='CCLLINK' name='discern'>

Metasymbol

reportname
A textual string representing the name of the Discern Explorer (CCL) report to be run. Context variables, when supported by the solution, can be substituted into the CCLLINK reportname.

prompts

A textual string representing the values that are passed into the prompt values associated with the report specified in reportname. These individual parameters need to be comma-delimited. Encryption of data is supported beginning with Discern Explorer version 8.4.4. Context variables, when supported by the solution, can be substituted into the CCLLINK prompts.

linkdestination

A numeric value representing the destination to direct the output of the link. The default value is 0.

ValueDescription

0

Used if the link is to be opened with its own window in Discern Report Viewer (DiscernReportViewer.exe). (To specify a particular window location, use APPLINK with Discern Report Viewer instead. See the APPLINK examples below.)

1

Used if the link is to open inside the current solution.

REPORTNAME is the Discern Explorer (CCL) program that is linked to, and it accepts the seven prompts below. The link opens inside the current solution and is given the title of Patient Summary Report.

  • PROMPT2 is a string.
  • The third argument is the person ID of the current patient.

  • The fifth argument is the encounter ID of the current patient.
  • The sixth argument is the person ID of the current user.
