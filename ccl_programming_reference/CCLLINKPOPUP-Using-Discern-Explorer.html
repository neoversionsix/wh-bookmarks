CCLLINKPOPUP is the routine used to open a HTML-supported, Discern Explorer (CCL) report in a new Internet Explorer window.

Note

For this routine to work, the following META tag must be included in the HEAD of the html file: <META content='CCLLINKPOPUP' name='discern'>

Syntax

CCLLINKPOPUP(reportname,prompts,target,features,replaceflag)

Metasymbols

reportname
A textual string representing the name of the Discern Explorer (CCL) report to be run.

prompts

A textual string representing the values that are passed into the prompt values associated with the report specified in reportname. These individual parameters need to be comma-delimited. Encryption of data is supported beginning with Discern Explorer version 8.4.4.

target

A textual string representing the name of the new session window. This name is used as the value for the TARGET attribute on a form or an anchor element.

features

A textual string that represents a list of features the window assumes. The features string contains a list of items separated by commas. Each item consists of an option and a value, separated by an equal (=) sign. For example: height=100, width=100. The following features are supported:

Feature

Description

height = number

Sets the height of the window in pixels.

left = number

Specifies the left position, in pixels. This value is relative to the upper-left corner of the screen.

top = number

Specifies the top position, in pixels. This value is relative to the upper-left corner of the screen.

width = number

Sets the width of the window in pixels.

resizable = yes/no/1/0

Specifies whether window can be resized.

status = yes/no/1/0

Specifies whether to add a status bar at bottom of window.

toolbar = yes/no/1/0

Specifies whether to display the browser toolbar.

replaceflag

A numeric value that specifies whether the URL creates a new entry or replaces the current entry in the window's history list. This parameter only takes effect if the _URL_is loaded into the same window.

ValueDescription
0Causes the URL to replace the current document in the history list.
1Creates a new entry in the history list.
