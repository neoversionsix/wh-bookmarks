CCLEVENT is the routine used to evoke special solution-specific events from within a web page.

Note

For this routine to work, the following META tag must be included in the HEAD of the html file: <META content='CCLEVENT' name='discern'>

Syntax

CCLEVENT(eventname)

Metasymbols

eventname

A textual string representing the name of the solution-specific event that is to be triggered. The following options are valid:

EVENT_EKS_OK

Specifies that the DiscernOutputViewer window navigates to the home page in the navigation history. This is valid in the context of MPages and Discern Notification (DiscernNotify.exe).

EVENT_NOTIFY_DELETE_HTMLONLYSpecifies that the Discern Notification message from which the event was evoked is deleted from the viewer. The <delete> key is disallowed, and only the HTML form containing this event can delete the message. This is valid only in the context of Discern Notification.

EVENT_NOTIFY_DELETE

Specifies that the Discern Notification message from which the event was evoked is deleted from the viewer. This is valid only in the context of Discern Notification.

EVENT_NOTIFY_MINIMIZE

Specifies that the Discern Notification viewer is minimized. This is valid only in the context of Discern Notification.

Examples

The below example illustrates the Discern Explorer (CCL) event EVENT_EKS_OK that is evoked when you click the button. This is a method to navigate back to the home page in the navigation history.

<a href='javascript:CCLEVENT("EVENT_EKS_OK")'>
Navigate Home</a>

 

The below example illustrates the CCL event EVENT_NOTIFY_DELETE that is evoked when you click the button. In Discern Notification, this event deletes the Discern Notification message from which the event was evoked after you perform another action.

<button onclick='javascript:CCLEVENT("EVENT_NOTIFY_DELETE");'>Button Delete and Minimize</button>

 

The below example illustrates the CCL event EVENT_NOTIFY_DELETE_HTMLONLY that is evoked when you click the button. The button deletes the message similar to the EVENT_NOTIFY_DELETE event; however, the message cannot be deleted by any other method.

 <button onclick='javascript:CCLEVENT("EVENT_NOTIFY_DELETE_HTMLONLY");'>Button Delete</button>
