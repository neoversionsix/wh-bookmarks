Aggregate functions are special group functions that can be used as a select expression or in Reportwriter FOOT sections. Aggregate functions are used to calculate values based on groups of rows. Without using the GROUP BY or HAVING clauses, one row is returned with the summary data.

The syntax of an aggregate function is shown below:

{aggregate_function (expression [WHERE qualification])
	["template; display_qualifier, display_format"]}

Valid aggregate functions are:

AVG( )
COUNT( )
MAX( )
MEDIAN( )
MIN( )
PERCENT( )
STDDEV( )
SUM( )
VARIANCE( )

The most common aggregate function is the COUNT( ) function. As a select expression, the COUNT( ) function can be used to quickly count the number of rows on a table. In the below example, the one value returned is the total number of rows on the CLINICAL_EVENT table.

SELECT	COUNT(*) FROM CLINICAL_EVENT GO

Other aggregate functions are based on actual numeric values in each row. In the example below, the one value returned is the average of all results that are greater than 0.0 that have been updated in the past six days.

SELECT   AVER = AVG(P.RESULT_VALUE_NUMERIC)
FROM
	PERFORM_RESULT   P

WHERE P.UPDT_DT_TM BETWEEN CNVTLOOKBEHIND("6,D") AND CNVTDATETIME(CURDATE, CURTIME3)
	AND P.RESULT_VALUE_NUMERIC > 0.0
GO
