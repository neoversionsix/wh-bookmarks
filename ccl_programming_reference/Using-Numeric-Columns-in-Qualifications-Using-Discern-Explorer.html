If you use a column that contains numeric data (instead of character data), do not embed in quotation marks the value to be compared to the column value. Punctuation (like commas or decimal points) also is unnecessary unless it is defined as part of the column.

To extract all encounter identification numbers greater than 3,000, enter the following command:

WHERE E.ENCNTR_ID > 3000

Notice that no quotation marks or commas are used because the ENCNTR_ID column is defined as a numeric column.

To extract all results where the result is equal to 4.5, enter the following command:

WHERE P.RESULT_VALUE_NUMERIC  = 4.5
