The CCLNEWSESSIONWINDOW command is the routine used to open a link in a new DiscernOutputViewer window.

Note

For this routine to work, the following META tag must be included in the HEAD of the html file: <META content='CCLNEWSESSIONWINDOW' name='discern'>

Syntax

CCLNEWSESSIONWINDOW(url,target,features,replaceflag,modalflag)

Metasymbols

url

A textual string representing the URL of the document to display in the new session window. If no URL is specified, a new window with about:blank is displayed.

target

A textual string representing the name of the new session window. This name is used as the value for the TARGET attribute on a form or an anchor element.

Value

Description

_blank

The URL is loaded into a new, unnamed window.

_parent

The URL is loaded into the current frame's parent. If the frame has no parent, this value acts as the value _self.

_search

The URL is opened in the browser's search pane.

_self

The current document is replaced with the specified URL.

_top

The URL replaces any framesets that might be loaded. If no framesets are defined, this value acts as the value _self

features

A textual string that represents a list of features the window assumes. The features string contains a list of items separated by commas. Each item consists of an option and a value, separated by an equals sign (=). For example, height=100,width=100. The following features are supported:

Feature

Description

height = number

Sets the height of the window in pixels.

left = number

Specifies the left position, in pixels. This value is relative to the upper-left corner of the screen.

top = number

Specifies the top position, in pixels. This value is relative to the upper-left corner of the screen.

width = number

Sets the width of the window in pixels.

resizable = yes/no/1/0

Specifies whether window can be resized.

status = yes/no/1/0

Specifies whether to add a status bar at bottom of window.

toolbar = yes/no/1/0

Specifies whether to display the browser toolbar.

scrollbars = yes/no/1/0

Specifies whether to display scroll bars.

fullscreen = yes/no/1/0

Specifies whether to display in full-screen mode. (Since the menu bar and task bar are hidden in full-screen mode, the only way to terminate the browser is to press ALT+F4. Therefore, a button or other alternate mechanism should always be provided when the user is ready to close the window.)

replaceflag

A numeric value that specifies whether the _URL_creates a new entry or replaces the current entry in the window's history list. This parameter only takes effect if the _URL_is loaded into the same window.

ValueDescription

0

Causes the URL to replace the current document in the history list.

1

Creates a new entry in the history list.

modalflag

A numeric value that indicates whether the window should be modal; the user cannot switch windows until the popup window is closed. When using nonmodal windows, take extra caution with the data displayed and the application context. For example, it may not be safe to load patient-specific data in a nonmodal window when the patient can be changed.

Handling Pop-Up Windows

Three levels of control handle pop-up windows:

  • Setting the HTML_POPUP_WINDOW_MODE in Preference Maintenance (PrefMaint.exe). This is a global setting that applies to the whole tab. If it is set to 1, the internal pop-up windows overrides the window.open() calls.
  • Setting the CCLOVERRIDEPOPUPWINDOW in the META tag of a page. This level allows for the control of session pop-ups per page. If you use this option, the global setting in first option must be set to 0. This allows the internal pop-up windows to override any window.open() call in the page.
  • Specifying the CCLNEWSESSIONWINDOW in the META tag of a page. This level controls the session pop-up per link. If this level of control is used, the global setting in the first option must be set to 0 and the CCLOVERRIDEPOPUPWINDOW in the second option must not be specified in the META tag. An example for this is a web page that needs only one link to open in a session pop-up window and other links to open in a regular browser pop-up window.

Examples

The example below illustrates opening the Oracle homepage in a new session window using CCLNEWSESSIONWINDOW. In this example, the Oracle homepage is titled Oracle in a new OutputViewer window with a height and width of 200. The page replaces the current entry in the history list.

<a href='javascript: CCLNEWSESSIONWINDOW("www.cerner.com","_blank","height=200,width=200",1,0)' >Cerner Homepage</a>
