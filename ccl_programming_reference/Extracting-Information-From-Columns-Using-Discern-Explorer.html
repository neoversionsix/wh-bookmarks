To select all columns from a table, you can use the wildcard * as your SELECT expression. 

SELECT * FROM PERSON GO

This command selects all the columns from the PERSON table. This type of simple select is very useful for identifying what type of data is in the columns of a table.

To select a specific column from a table, use the table and column name in your SELECT statement. For example:

SELECT PERSON.NAME_FULL_FORMATTED FROM PERSON GO

All columns need to be referenced to the table on which they are located. The table name PERSON and the column name NAME_FULL_FORMATTED are connected using a period.
The following output is from this query:

NAME_FULL_FORMATTED
Brown, Joseph
Doe, William
Smith, Sue
Smith, Jane
Jones, Steve
Brown, Mary

You can select several columns at once using the SELECT command. The sample query shown below selects the following information from the ENCOUNTER table:

  • Encounter identification number
  • Person identification number
  • Location code
  • Arrival date and time

Notice that a comma separates each column name, and a period connects each column name to a table name. The order in which columns are selected is the order they are displayed or printed.

SELECT	ENCOUNTER.ENCNTR_ID,
	ENCOUNTER.PERSON_ID,
	ENCOUNTER.LOCATION_CD,
	ENCOUNTER.ARRIVE_DT_TM
FROM	ENCOUNTER
GO
