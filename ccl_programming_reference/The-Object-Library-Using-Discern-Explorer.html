The object library is the repository of executable programs that have been created using the CREATE PROGRAM command. When you use the CREATE PROGRAM command in your source code, and then include that source code, an object is created and stored in the object library. An object is an executable program that is stored in computer language, which can be executed without recompiling the file.

Note

The object library is the same file that contains the Discern Explorer Data Dictionary (CCLDIR:DIC.DAT).

Objects are included in the object library using the %INCLUDE or %RUN directives.
