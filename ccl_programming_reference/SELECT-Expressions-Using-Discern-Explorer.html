SELECT expressions can be either columns from the database or user-defined expressions that have a value assigned to them. See User-Defined Select Expression for more information about user-defined expressions.

Below is an example query. 

SELECT	PR.NAME_LAST_KEY,
	PR.BIRTH_DT_TM
FROM	PERSON PR
GO

The following is an example of the output displayed from the previous query:

NAME_LAST_KEY	BIRTH_DT_TM
BROWN	        03/23/63
JONES	        09/25/95
STROUD	        04/17/45
SMITH	        06/29/81
FITZPATRICK	12/06/68
SLIMMER	        05/12/97
SUMMERS	        11/23/73

Note

When using multiple SELECT expressions, separate the expressions with commas. Do not place a comma after the last SELECT expression.
