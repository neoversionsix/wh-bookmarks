The RECORD command is used to create a Record Structure definition.

The FREE RECORD command is used to destroy a Record structure definition. All memory is deallocated.

You can copy one Record structure to another by defining each structure, setting the record items, and then setting one structure equal to the other.

RECORD TEMP_1
( 1 TEST_ID = F8 ) GO

RECORD TEMP_2
( 1 TEST_ID = F8 ) GO

SET TEMP_1->TEST_ID = 1 GO
SET TEMP_2 = TEMP_1 GO

The MOVEREC( ) and MOVERECLIST( ) functions can be used to move values from one Record structure to another.

The LOCATEVAL( ) and LOCATEVALSORT( ) functions can be used to search a Record structure for a specific value.

The ALTER2( ), ALTERLIST( ), or ALTER ( ) functions are used to allocate memory for Record Structure lists.

The INITREC( ) function is used to reset a Record structure. It deallocates all memory for the Record Structure lists but retains the Record Structure definition.

The EXPAND( ) function can be used to create a qualification in a query that compares the values from a Record Structure list to a field on a Relational Database Management System

(RDBMS) table.
