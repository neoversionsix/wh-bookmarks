You can sort and display data in the sequence you choose by using the ORDER clause in the SELECT command. Sorting is performed after data is extracted, but before it is displayed. The syntax of the ORDER clause is shown below:

[ORDER	expression [DESC] {,expression [DESC] }]

The ORDER clause follows the WHERE clause in the SELECT command, or the FROM clause if the WHERE clause is not used. Rows are sorted in the order in which they are listed in the ORDER clause. The sample query below uses the ORDER clause to sort the information extracted from the PERSON table by the last name.

SELECT	P.NAME_FULL_FORMATTED,
	P.BIRTH_DT_TM,
	P.SEX_CD
FROM	PERSON P
ORDER	P.NAME_LAST_KEY
GO

The following example is the output from the query:

NAME_FULL_FORMATTED    BIRTH_DT_TM    SEX_CD
Brown, Alice           08/06/77     15075.00
Brown, Jane            06/01/54     15075.00
Brown, John            02/05/59     15080.00
Brown, Joseph          06/27/66     15080.00
Brown, Mary            09/20/68     15075.00
Brown, Steve           09/18/79     15080.00
Brown, Sue             08/06/77     15075.00
Brown, William         08/07/91     15080.00

The ORDER clause displayed in the query above uses the NAME_LAST_KEY column to organize the information alphabetically by the person's name. This column is not one of the columns from which data is extracted. You do not need to include the sort column in the SELECT command unless you want the column included in the result display.

Discern Explorer sorts information in ascending order unless you specify otherwise by entering "DESC" (for descending) after the name of the column that is used for sorting. For example, to sort the information in descending order by NAME_LAST_KEY, use:

ORDER	P.NAME_LAST_KEY DESC

You can also use the column number instead of using the column name if you are ordering on a column from your select. This has to be used for Select Union queries that use an order by clause. The following example uses the PERSON_ID to order the output in descending order:

SELECT 	P.NAME_FULL_FORMATTED,
	P.PERSON_ID,
	P.SEX_CD
FROM 	PERSON P
ORDER   	2 DESC
GO
