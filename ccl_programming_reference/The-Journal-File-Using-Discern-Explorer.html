As you enter commands at the command line during a Discern Explorer session, all your keystrokes except directives are saved in a journal file. By capturing the keystrokes you enter, the journal file enables you to save the keystrokes as a Discern Explorer file, even if you did not intendÂ to do so originally. For example, if you use Discern Explorer to create an ad hoc query that extracts patient names. If you decide that you want to reuse the particular sequence of Discern Explorer commands, you can access the journal file using the %JOURNAL directive. You then can save the commands as a Discern Explorer script file, which can be accessed and edited like any other script. When accessing the journal file, it is not necessary to specify a file name. Enter %J and press ENTER. Discern Explorer opens the journal file using the default editor.

To copy the journal file to a new file, use the following editor commands.

For VMS/EDT

  1. Press CTRL+ Z. The system displays the * prompt.
  2. To return to the journal file, press C at the * prompt.
  3. To copy the journal file, enter EXIT file_name and press ENTER on the numeric keypad. The system copies the file and returns to the Command Line mode.
  4. To close the file without saving the changes, enter PF1 Q.

For UNIX/Emacs

  1. To copy the journal file, press PF1 W. Enter the new file name at the prompt. The system copies the file and returns to the Command Line mode.
  2. To close the file without saving the changes, enter PF1 Q.

Note

When you exit or quit the journal file, all commands in the file will be executed.

Enter %CLEAR (%C) at the Discern Explorer command line to turn off journaling if it is on, and on if it is off (%C is a toggle). This also clears the journal file of keystrokes from the current session.
The journal file is re-created automatically each time you use Discern Explorer. Each time you access a journal file (by entering "%J" and pressing <Return>) it relates to the current session only.

The journal file is assigned a filename that is a combination of the letters HIS, the process name (usually your answer back if you are using an Millennium Platform sign-on from the terminal from which the Discern Explorer session was run), and the letters CCL for the extension. When you exit Discern Explorer, the journal file is deleted.
