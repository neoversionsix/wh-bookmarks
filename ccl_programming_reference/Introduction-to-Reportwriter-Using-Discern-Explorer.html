The Reportwriter clauses are an extension of the Discern Explorer SELECT command. These clauses cannot be used as stand-alone commands. Reportwriter clauses are used process the result set data that is returned by the query portion of the SELECT command. Using Reportwriter clauses you can define page headers and footers, headings on one or more pages, summary pages, group titles, and so on.

Discern Explorer applies default format characteristics unless you define the appearance of your output using Reportwriter clauses. While the defaults are adequate for many purposes, you probably want to improve the appearance of your reports. With Reportwriter, you can make your reports more readable, more clearly organized, and more useful. With headings defined by you, and report sections clearly marked, the reader of your report can more easily navigate through it, locating specific parts of interest. The Reportwriter clauses give you complete control over where data is printed, when data is printed, or even whether data is printed. You can use Reportwriter clauses to keep track of running totals, compute averages, minimums, maximums, and set flags to perform, or not perform, certain functions.
