The HEAD REPORT clause also is used to define macros to be called throughout the report. To illustrate the use of a macro, consider the following example. The COL_HEADS macro is defined in the HEAD REPORT section and then called in the HEAD PAGE and HEAD NURSE_UNIT clauses. Some checks would need to be added to prevent the headings from printing twice at the top of the page when a nursing unit starts at the top of the page.

HEAD REPORT
MACRO (COL_HEADS)
	COL 0 "Name:"
	COL 50 "Sex:"
	COL 60 "Birth Date:"
	COL 75 "Room-Bed:"
	ROW +1
	ENDMACRO
HEAD PAGE
	COL_HEADS
HEAD NURSE_UNIT
	ROW +1
	COL  0 "Nursing Unit:"
	COL +2 NURSE_UNIT
	ROW +1
 	COL_HEADS
