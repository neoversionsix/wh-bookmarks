The HEAD REPORT clause has three uses:

  • Initialize expressions.
  • Define macros.
  • Define information to be printed at the beginning of the report, such as a report title.

The syntax of the HEAD REPORT clause is shown below.

[HEAD REPORT
	report_item {,
	report_item }]

Keep these points in mind when you use the HEAD REPORT clause:

  • The HEAD REPORT clause is optional.
  • The SELECT command can include only one HEAD REPORT clause.
  • If the HEAD REPORT clause is used, at least one report item must be used.
