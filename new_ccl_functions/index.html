SET commands are used broadly within Discern Explorer. In Screen applications, SET commands control user options that are available during the execution of a Screen application. SET commands can be used to create help on a field, validate data entered by the user, provide documentation on a field, branch to a label when the HOME key is pressed, and other similar commands.
