CNVTB36B10( ) converts a base-36 string to a base-10 string with the maximum length defined by length. This function produces values primarily used in bar coding.

Syntax

CNVTB36B10(base36_string,length[,mode])

Metasymbols

base36_string

An expression that is a base-36 string.

length

An integer that defines the number of bytes in a string.

mode

Mode is used to flex the base-36 sequence. Modes 2 and 3 were added in Discern Explorer versions 8.13.3 and 9.01.3.

ModeDescription
0Use the base-36 sequence of ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789. Each group of 6 digits is converted separately. This mode is the default as is primarily used for bar codes.
1

Use the base-36 sequence of 0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ. Each group of 6 digits is converted separately. This mode is primarily used for bar codes.

2

Use base-36 sequence of ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789 and convert from base-36 to base-10 as one value.

3Use base-36 sequence of 0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ and convert from base-36 to base-10 as one value.


Examples

In this example, C is set to 023001019.

C = CNVTB36B10("ANY9ZH",9)
