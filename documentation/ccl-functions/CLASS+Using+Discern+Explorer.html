The CLASS () function is used to return the description of class items or instance items. CLASS was introduced in Discern Explorer version 8.9.0.

Syntax

CLASS(class_instance,class_info_id|class_instance_info_id,[mode])


Metasymbols

class_instance_info_id

Used with mode of 0 or 1.

1

i_name

2

i_full_name

3

i_mangled_name

4

i_namespace

5

i_lev

class_info_id 

Used with mode of 2 or 3.

1

c_refcnt

2

c_prglev

3

c_final

mode

0

Returns instance item description.

1

Returns instance item name and description.

2

Returns class description.

3

Returns class item name and description.
