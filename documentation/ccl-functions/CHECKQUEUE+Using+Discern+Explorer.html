You can use CHECKQUEUE to determine whether a given print queue name is valid. Returns 1 if the queue is valid and a 0 if it is not.

Syntax

CHECKQUEUE(queue_name[, dms_service_name])

Metasymbol

queue_name

The name of the print queue to check.

dms_service_name

The Digital Media Services name. Allows function to return DMS name if DMS is enabled.

Examples

The following example will set the variable ISQUEUE to 1 if "P350" is a valid queue or 0 if it is not.
SET VAR1 = "P350"
SET ISQUEUE = CHECKQUEUE(VAR1)
