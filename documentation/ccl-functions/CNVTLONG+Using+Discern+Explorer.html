The CNVTLONG( ) function converts a character or alphanumeric expression to a long64 integer. A long64 integer is a Discern Explorer W8 data type. The CNVTLONG( ) function was introduced in version 8.9.0 of Discern Explorer.

Syntax

CNVTLONG(expression)

Metasymbol

expression
A character or alphanumeric expression.

Examples

In the following example, long_int is set equal to 1234567890123456.

declare long_int = w8 
set long_int = cnvtlong("1234567890123456")
