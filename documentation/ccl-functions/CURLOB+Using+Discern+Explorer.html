CURLOB allows you to pass a BLOB or CLOB parameter to a PLSQL function. It returns the expression but if it is used as bind parameter to PLSQL function, it binds the expression based on the mode.

CURLOB was introduced in version 8.10.0 of Discern Explorer.

Syntax

CURLOB(expression, mode)

Metasymbol

expression
A BLOB or CLOB expression.

mode

Mode

Description

1

Pass as char

2

Pass as raw

3

Pass as longchar

4

Pass as longrawchar

Examples

The following example uses CURLOB().

drop program ccl_curlob_exam go
create program ccl_curlob_exam 
record rec1 (
  1 blob = gvc )

set rec1->blob = 'testing this line'

declare hash(p1=vc,p2=i4) = c32 with sql = 'DBMS_CRYPTO.HASH'
declare toraw(p1=vc)=null with sql = 'UTL_RAW.CAST_TO_RAW'

select into 'NOFORMS' hash(toraw(curlob(rec1->blob,4)),2) from dual

select into 'NOFORMS' hash(curlob(rec1->blob,4),2) from dual

end go
