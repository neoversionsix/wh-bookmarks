BNOT () returns an integer expression that is the result of each bit in the base-2 number of the integer being reversed. This means that the BNOT( ) function changes all one bits to zeros, and zero bits to ones. Additionally, BNOT( ) changes the sign bit. If the value of the sign bit is zero, the integer is positive. If the sign bit value is one, the integer is negative.

Syntax

BNOT(integer)

Metasymbols

Integer
Any whole number.

The key to understanding how the BNOT( ) function works is to first convert the integer to base-2.

In base-2, numbers are represented by placing a zero or one in a bit number position and a subscript value of 2. In the base-10 system, the number base in which we usually count, we count from one to nine, carry the one, and start over at zero. The below examples display conversions from base-10 to base-2.

The assigned bit that is not displayed changes from zero to one, making this a negative number. When you convert this back to base-10, the following result is displayed:

The BNOT() function returns the following values for the integer:  

  • The negative of the original numbers minus one if the original number was positive.
  • The positive of the original number minus one if the original number was negative.
