CURPROG returns the program name for a program level. It returns a blank space if the program level is not in the current program execution path, otherwise, the program, or subprogram name for that level is returned. The program level must be between 1 and the current level.

Syntax

CURPROG(level)

Metasymbol

level

An integer value representing the program level.

Examples

  drop program ccl_test_curproga:dba go
  create program ccl_test_curproga:dba
      execute ccl_test_curprogb
   end go

   drop program ccl_test_curprogb:dba go
   create program ccl_test_curprogb:dba
      execute ccl_test_curprogc
   end go

   drop program ccl_test_curprogc:dba go
   create program ccl_test_curprogc:dba
      declare progname = vc
      declare num = i4
      set num = 1
      while (num > 0)
          set progname = curprog(num)
          if (progname != " ")
             call echo(build("program name for level:",num," = ", progname))
             set num = num+1 
          else
             set num = 0  ;stop loop
          endif
      endwhile
   end go
