CURLOCALE returns the value of a locale name for the environment. Before Discern Explorer version 8.11.5, the maximum length of the return value is 132 characters. Discern Explorer version 8.11.5, increased the maximum length of the return value to 200 characters.

Syntax

CURLOCALE(locale_string)

Metasymbols

Locale__string

The locale string as defined in the CCLSTARTUP_LOCALE program for the environment. The following table lists some possible locale strings.

Examples

The following example echoes the CURLOCALE values for hours and full week names.

SET HH = CURLOCALE("HOURS")
CALL ECHO(HH)
SET WFULL = CURLOCALE("WEEKFULL")
CALL ECHO(WFULL))
