The BUILD( ) function returns an expression concatenated, with trailing blanks removed; integer and real values are treated as strings. The function acts as a concatenate function for combining character expressions with numeric expressions.

Discern Explorer version 8.12.3 increases the semantic check length from 20 to 24 characters when the BUILD() function is used to pass a parameter with an F8 datatype as an argument to a subroutine call. 

Syntax

BUILD(expression{,expression})

Metasymbols

Expression
A value of any data type.

Examples

In this example, the result is ABC12DEFG3456HIJ.

EX=BUILD("ABC ",12," DEFG",3456," HIJ")
