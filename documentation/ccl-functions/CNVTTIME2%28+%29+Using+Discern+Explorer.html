The CNVTTIME2() function converts a Character time into a Oracle Internal System time in 10 millisecond units. Each metasymbol must be of type Character. The time_input_format can contain HH, MM, SS, and CC to reflect the input format of the expression that is being converted. Punctuation, such as colons, periods, or spaces can be included to permit those characters in the character_time expression.

Syntax

CNVTTIME2(character_time , time_input_format)

Metasymbols

character_time
A character description of a time.

time_input_format
The format of the time that is being converted. The valid format characters are listed below

Character

Description

HH

hours

MM

minutes

SS

seconds

CC

hundredth of seconds

other

special characters, such as colons, periods, or spaces


Examples

In the following example, the output of the Call Echo() command is 15-MAR-2008 23:10:20.00.

SET DATESTRING = "20080315"
SET TIMESTRING = "23:10:20:00"
SET DT_TM = CNVTDATETIME(CNVTDATE2(DATESTRING,"YYYYMMDD"),CNVTTIME2(TIMESTRING,"HH:MM:SS:CC"))
CALL ECHO(FORMAT(DT_TM, ";;Q"))


In the following example, the output of the Call Echo() command is 15-MAR-2008 23:10:00.00.

SET DATESTRING = "20080315"
SET TIMESTRING = "2310"
SET DT_TM = CNVTDATETIME(CNVTDATE2(DATESTRING,"YYYYMMDD"),CNVTTIME2(TIMESTRING,"HHMM"))
CALL ECHO(FORMAT(DT_TM, ";;Q"))
