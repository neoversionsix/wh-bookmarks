The DATETIMEPART() function allows you to return the year, month, day, hour, or minute part of a datetime value as an integer.

When Coordinated Universal Time (UTC) is turned on and DATETIMEPART() is used; outside of a SELECT, in the selection list, or in a reportwriter section of a SELECT, the DATETIMEPART() function assumes the datetime value is in UTC and converts it to a local datetime before returning the part. If UTC is turned on and DATETIMEPART() is used in a query in a way that causes it to be processed at the RDBMS level, the part is returned from the UTC datetime value before it is adjusted to the local time zone.

DATETIMEPART() cannot be used in a nested Select command.

Syntax

DATETIMEPART(datetime, part_type)

Metasymbols

datetime

A datetime value

part_type

Determines the part to be returned from the datetime value

Part_typeDescription 

1

year component

2

month component

3

day component

4

hour component

5

minute component

Examples

The following example echos 2016:

call echo(datetimepart(cnvtdatetime("20-Jun-2016 13:45:00"),1)) go

The following example echos 6:

call echo(datetimepart(cnvtdatetime("20-Jun-2016 13:45:00"),2)) go

The following example echos 20:

call echo(datetimepart(cnvtdatetime("20-Jun-2016 13:45:00"),3)) go

The following example echos 13:

call echo(datetimepart(cnvtdatetime("20-Jun-2016 13:45:00"),4)) go

The following example echos 45:

call echo(datetimepart(cnvtdatetime("20-Jun-2016 13:45:00"),5)) go

The following example echos year is        2016:

call echo(build2("year is ", datetimepart(cnvtdatetime("20-Jun-2016 13:45:00"),1))) go

The following example echos 2016:

set year = datetimepart(cnvtdatetime("20-Jun-2016 13:45:00"),1) go
call echo(year) go

The following example returns rows from the Person table that were updated on the 15th day of any month in the past year:

SELECT
    P.UPDT_DT_TM
FROM
    PERSON   P
WHERE P.UPDT_DT_TM BETWEEN CNVTLOOKBEHIND("1,Y") AND CNVTDATETIME(CURDATE, curtime3)
    AND datetimepart(P.UPDT_DT_TM,3) = 15
WITH NOCOUNTER, SEPARATOR=" ", FORMAT

The following example will generate errors because DATETIMEPART() cannot be used in nested Select commands:

SELECT
    O.MONTH
FROM
    OMF_DATE   O
WHERE O.MONTH = 
    (select o2.month
     from omf_date o2
     where datetimepart(o2.updt_dt_tm,2) = o2.month)
WITH NOCOUNTER, SEPARATOR=" ", FORMAT
