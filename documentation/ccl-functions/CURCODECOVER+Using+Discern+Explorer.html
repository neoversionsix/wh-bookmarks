The CURCODECOVER( ) function displays the code coverage information from within a program. It is required that the program be compiled in debug mode and that the COST and CODECOVER traces be turned on. The code coverage information is useful for determining the lines in a program that were executed when testing the program. Lines that were executed were covered by the test.

The code coverage information is displayed in the following format:

COVER:: Prg(program name) Lev(program level) Trk(track_mode) Tst(test number, process name) Row(percent covered)(N non-executable lines :U executable lines not covered :C executable lines covered :T executable lines uncovered + executable lines covered)


COVER:: line_type:line_number(s) [#cnt]

line_type

N: Non-executable line. Includes comment lines or clauses such as ENDIF, ENDFOR, ENDWHILE, ENDCASE, CREATE PROGRAM, PROMPT.
U: Uncovered executable line.
C: Covered executable line.

line_number(s)

One or more line numbersÂ from the listing of the source code file compiled in debug mode.

#cnt
Only shown if line was executed more than once.

Syntax

CURCODECOVER(mode)

Metasymbols

mode

Option

Description

1

Print the code coverage information to the ccloutput file

2

Begin the code coverage

3

End the code coverage

Examples

Compiling the following code in debug mode:
DROP PROGRAM test_codecover GO
CREATE PROGRAM test_codecover
 prompt
 "Output to File/Printer/MINE" = "MINE"
with OUTDEV

set curcodecover = 1
set stat = curcodecover(2)
select into "NL:"
p.person_id from person p with maxrec = 5
set stat = curcodecover(3)
set stat = curcodecover(1)

set curcodecover = 2
set stat = curcodecover(2)

SELECT INTO $OUTDEV
 P.PERSON_ID
 , P.NAME_FULL_FORMATTED
 , rec_ind = decode(p.seq,1)
FROM
 PERSON   P
WHERE  P.PERSON_ID < 100.0


Head Report
 dcnt = 0
 ROW 2 COL 13 "This is the Report Header"

 ROW 3 COL 13  curqual
 ROW 3 COL 29 IF ( curqual > 0 )
 "Records qualified"
 ELSE
 "No records qualified"
 ENDIF
 ROW + 2

Detail
 if ((ROW + 2) >= maxrow)  break endif
 dcnt = dcnt +1
 COL 5   curqual
 col +1  P.PERSON_ID
 COL +1  P.NAME_FULL_FORMATTED
 ROW + 1

Foot Report
 COL 5  IF ( dcnt = 0 )
 "No Records Qualified"
 ENDIF
 ROW + 1
 COL 13  "This is the report footer"


WITH MAXREC = 100, NULLREPORT, maxcol = 200

set stat = curcodecover(3)
set stat = curcodecover(1)

END
GO
