The CNVTALPHANUM( ) function allows you to return a string stripped of nonalphanumeric characters. Only alphanumeric characters are retained; any special characters are removed.

Syntax

CNVTALPHANUM(expression >[,mode])

Metasymbol

expression
A value of character data type.

mode
An integer value representing the bitwise BOR of the following:

Integer

Description

Value Range

1

Keep number

0 through 9

2

Keep alpha with multinational characters

a through z
A through Z
multinational characters with ASCII ordinal values >=128

4

Keep punctuation

+ - . ,

8

Keep all punctuation

! through / 
: through @ 
[ through `

16

Keep pattern matching characters

* ?

32

Keep pattern matching character

\

64

Keep alpha without multinational characters

a through z
A through Z
but not multinational characters with ASCII ordinal values >=128

If not specified, the mode defaults to 3. The bitwise BOR of the mode value results in keeping those options. For example:

3, the bitwise BOR of 1 and 2 retains number and alpha.
7, the bitwise BOR of 1, 2, and 4 retains (+|-|.|,) number and alpha.

Examples

In this example, A is set to "ABCDEF01200".
A = CNVTALPHANUM("ABC-DEF-012-00")
In this example, A is set to "OBrien".
A = CNVTALPHANUM("O'Brien")
