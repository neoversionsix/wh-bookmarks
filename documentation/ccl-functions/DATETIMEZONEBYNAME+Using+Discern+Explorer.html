The DATETIMEZONEBYNAME( ) function returns the time zone index (integer)Â when given the time zone name. It returns the time zone index if a valid name, else it returns zero (0). DATETIMEZONEBYINDEX( ) is the converse of DATETIMEZONEBYNAME( )

Syntax

DATETIMEZONEBYNAME(timezone_name)

Metasymbol

timezone_name

A text expression representing the name of the time zone.

Note

Oracle has adopted a Windows standard that has defined the time zone names and their associated index numbers.

Examples

In the following example, the variable INDEX is set to 91:

SET INDEX = DATETIMEZONEBYNAME("CST") GO

In the following example, the variable INDEX is set to 75:

SET INDEX = DATETIMEZONEBYNAME("America/Chicago") GO
