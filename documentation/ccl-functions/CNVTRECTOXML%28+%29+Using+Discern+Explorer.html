The CNVTRECTOXML( ) function converts a Discern Explorer record structure to an XML record structure. The XML record structure is returned as a VC data type.

For information about writing the Discern Explorer record structure to the displayer, default log or listing file, or to a user-defined file as an XML stream, see CALL ECHOXML.

Syntax

CNVTRECTOXML(record_name[,case_mode[,datemode[,flexmode]]])

Metasymbols

record_name
The name of a Discern Explorer record structure.

case_mode

Introduced in Discern Explorer versions 8.14.0 and 9.02.0. The only mode for the initial release is 0. Additional options may be supported in future releases.

ModeDescription
0The default mode.

date_mode

Introduced in Discern Explorer versions 8.14.0 and 9.02.0 to allow controlling the use of Date() for ISO dates.

ModeDescription
0Use Date(iso_date) for DQ8 or DM12 (default).
1Use iso_date without Date() for DQ8 or DM12.
2Use Date(iso_date) for DQ8 or DM12 (default), and convert the Coordinated Universal Time (UTC) to the local time, regardless of the CURUTC.
3Use iso_date without Date() for DQ8 or DM12, and convert the UTC to the local time, regardless of the CURUTC.

flex_mode

Introduced in Discern Explorer versions 8.14.0 and 9.02.0. The only mode for the initial release is 0. Additional options may be supported in future releases.

ModeDescription
0The default mode.


Examples

The example below creates and populates a Discern Explorer record structure named REC1. The contents of the record structure are then written as an XML stream to a variable named REC_XML.

drop program CCL_CnvtRecToXml go
create program CCL_CnvtRecToXml
record rec1(
  1 qual[*]
    2 f1 = i4
    2 qual2[*]
      3 f2 = i4
      3 f3 = vc)
declare rec_xml = vc

set stat = alterlist(rec1->qual,3)
for (num1=1 to size(rec1->qual,5))
    set rec1->qual[num1].f1 = num1
    set stat = alterlist(rec1->qual[num1].qual2,1)
    set num2 = 1
    set rec1->qual[num1].qual2[num2].f2=100+num1
    set rec1->qual[num1].qual2[num2].f3=build("test:",num1,":",num2)
endfor
call echorecord(rec1)
;convert ccl record to xml string
set rec_xml = CnvtRecToXml(rec1)

END GO
