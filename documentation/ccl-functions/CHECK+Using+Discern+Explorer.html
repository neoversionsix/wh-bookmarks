The CHECK() function replaces all characters that are less than the check_char with blank characters. If check_char is omitted, the default of blank is used. If check_char2 is used, all characters greater than it are replaced with a blank character.

Syntax

CHECK(expr [,check_char, check_char2]])

Metasymbols

expr

Any character expression.

check_char

The character to check for. Characters with an ASCII ordinal value less than check_char are converted to a space.

check_char2

The character to check for. Characters with an ASCII ordinal value greater than check_char2 are converted to a space.


Before Discern Explorer version 8.12.3, check_char and check_char2 were limited to characters with an ASCII ordinal value of 127 or less. Using a character with an ASCII ordinal value greater than 127 causes the entire expression to be converted to spaces. Discern Explorer version 8.12.3 allows extended ASCII characters with ordinal values greater than 127 for check_char and check_char2.

Examples

The following example replaces abc with spaces, and returns the string defghijk:

CALL ECHO(CHECK("abcdefghijk","d")) GO

This following example returns the string W L OM TO RN R:

CALL ECHO(CHECK("WELCOME TO CERNER","L")) GO

The following example replaces all characters that are less than a carriage return (char(13)) with a blank character, and all characters that are greater than a tilde (~), char(126), with a blank character.

 SET VAR = CHECK(STRING_VAR,CHAR(13),CHAR(126))
