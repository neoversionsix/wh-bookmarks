This function is used to determine if an object exists in the object library and if the user has access to it.

Returns 0 if object does not exist
Returns 1 if object exists and the user does not have permission to access it
Returns 2 if object exists, and the user does have permission to access it.

Syntax

CHECKDIC(object_name, object_type, group_number)

Metasymbols

Object_name

The name of the object in the Discern Explorer object library. Must be in uppercase.

Object_type

The type of the object in the Discern Explorer object library. Must be in uppercase.

T = Table
V = View
P = Program
E = Ekmodule
D = Database
M = Menu
A = table field Attribute


Group_number

The CCLGROUP number for the object.  0 for DBA, 1 through 99 for other groups.

Examples

SET FIELD_EXISTS = CHECKDIC("PERSON.PERSON_ID", "A", 0)
SET PROG_EXISTS  = CHECKDIC("CCLPROT", "P", 0)
SET PROG_EXISTS  = CHECKDIC("MY_PROG", "P", 1)
SET TABLE_EXISTS = CHECKDIC("PERSON", "T", 0)
SET OBJ_EXISTS = CHECKDIC(CNVTUPPER($1), CNVTUPPER($2), $3)
