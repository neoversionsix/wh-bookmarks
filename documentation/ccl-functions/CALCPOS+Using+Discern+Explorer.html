The CALCPOS () function returns the DIO position command for the integer metasymbols horizontal and vertical.

Syntax

CALCPOS(horizontal, vertical)

Metasymbols

horizontal
Any whole number that represents the horizontal DIO position.

vertical
Any whole number that represents the vertical DIO position.

Examples

In the below example, the final printed result is {POS/12/9}.

SMG_VARX = 5
SMG_VARY = 6
CALL PRINT(CALCPOS(7 + SMG_VARX, 3 + SMG_VARY))
{"serverDuration": 593, "requestCorrelationId": "b7e0bb050f0dd460"}
