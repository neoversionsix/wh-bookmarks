The CNVTTIME() function  accepts a Number of Minutes or Datetime value and returns an integer representing time in 24-hour format of the form HHMM. The conversion can be performed in reverse by calling the CNVTMIN() function. A display option or the Format() function can be used to have the time displayed in Standard Time format using A.M. or P.M.

Syntax

CNVTTIME(time)
CNVTTIME(datetime)

Metasymbols

time

An expression of time in number of minutes.

datetime

The value that represents a date and time. RDBMS fields with names that end in _DT_TM store Datetime values. Many Datetime functions like CNVTDATETIME() can be used to return a Datetime value.

Examples

In the example below, if the user-defined expression ADM_TIME equals 780 minutes, TIME is set to 1300, 13:00 or 01:00 PM.

TIME = CNVTTIME(ADM_TIME)

or

TIME = CNVTTIME(ADM_TIME) "HH:MM;;M"

or

TIME = CNVTTIME(ADM_TIME) "HH:MM;;S"

In the example below, if ORDER_DT_TM is 01-JAN-1998 10:50:59, TIME is set to 1050.  If ORDER_DT_TM is 01-JAN-1998 09:30:59, TIME is set to 930.

TIME = CNVTTIME(O.ORDER_DT_TM)
