Returns memory information for the current process.

Syntax

CURMEM(mode)

Metasymbol

mode

An integer value that indicates the memory item to return.

OptionItemDesc
1rlim_curSoft rlimit
2rlim_maxHard rlimit
3arenaNon-mmapped space allocated in bytes
4ordblksNumber of free chunks
5smblksNumber of free fastbin blocks
6hblksNumber of mmapped regions
7hblkhdSpace allocated in mmapped regions in bytes
8usmblksMaximum total allocated space in bytes
9uordblksTotal allocated space in bytes


Examples

The below example echoes the value of all the memory items for the current process.

For(lcnt = 1 to 9)
    call echo(curmem(lcnt))
endfor
