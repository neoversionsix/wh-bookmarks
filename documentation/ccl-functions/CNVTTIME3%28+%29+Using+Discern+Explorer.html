The CNVTTIME3( ) function converts a time expression to hundredths of seconds.

Syntax

CNVTTIME3(time,mode)

Metasymbols

time
An expression that represents a time in a specified format.

mode

ModeDescription

1

Time is in HHMM format.

2

Time is in HHMMSS format.

3

Time is in HHMMSSCC format.

Examples

In the following example, hhmm_var is set to 3780000.0, hhmmss_var is set to 5254900.0, and hhmmsscc_var is set to 6641376.0:

set hhmm_var = cnvttime3(1030,1)
set hhmmss_var = cnvttime3(143549,2)
set hhmmsscc_var = cnvttime3(18265376,3)

In the following example, if the current time is 2:45 PM, hun_var is set to 5310000.0:

hun_var = cnvttime3(curtime,1)
