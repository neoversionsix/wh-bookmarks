The COPYREC( ) function copies a record structure definition or record structure definition and data depending on mode. The record_name_from must exist and the record_name_to must not exist. A record structure is defined using the RECORD command. COPYREC() was introduced in version 8.10.0 of Discern Explorer.

Syntax

COPYREC(Record_Name_From, Record_Name_To[,Mode])

Metasymbols

record_name_from
The name of a record structure that previously has been defined using the RECORD command.

record_name_to
The name of a record structure that has not been defined using the RECORD command.

mode

ModeDescription
0copy record structure definition only (default).
1copy record structure definition and data.

Examples

The following example creates RecB, giving it the same definition as RecA. RecC is also created with the same definition and data as RecA.

drop program ccl_test_copyrec go
create program ccl_test_copyrec

record RecA (
  1 List [*]
    2 Int_Fld = i4
    2 Flt_Fld = f8
    2 VC_Fld = VC
   )

set stat = alterlist(RecA->List,10)
for(lcnt = 1 to 10)
        set RecA->List[lcnt].Int_Fld = lcnt
        set RecA->List[lcnt].Flt_Fld = cnvtreal(lcnt)
        set RecA->List[lcnt].VC_Fld = cnvtstring(lcnt)
endfor
call echorecord(RecA)

set stat = copyrec(RecA,RecB,0)
call echorecord(RecB)

set stat = copyrec(RecA,RecC,1)
call echorecord(RecC)

end
go
