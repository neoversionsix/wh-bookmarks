The BTEST( ) function returns an integer expression that is a one if the bit_number position of the base-2 conversion of the integer is a one. Likewise, zero is returned if the bit_number position of the base-2 conversion of the integer is a zero. The bit_number is the position of a bit in a field.

Syntax

BTEST(integer,bit_position)

Metasymbols

Integer
Any whole number.

Bit_position
The number of the integer position, working from right to left.

The key to understanding how the BTEST( ) function works is to first convert the integer being tested into a base-2 number.

In base-2, numbers are represented by placing a zero or one in a bit number position and a subscript value of 2. In the base-10 system, the number base in which we usually count, we count from one to nine, carry the one, and start over at zero. The below examples display conversions from base-10 to base-2.

The BTEST( ) function identifies if the conversion of an integer to base-2 results in a one being present in the given bit_number position. If a one is present in the bit_number position, then the result of the BTEST( ) function is a one. If not, then a zero is returned.

Examples

The following example displays the integer five and a bit_position of two:

RESULT=BTEST(5,2)

510=000001012

Looking from right to left, you can see that 000001012 has a one in the zero and first bit_number positions. In the second position, one is displayed. Therefore, the result is one.
