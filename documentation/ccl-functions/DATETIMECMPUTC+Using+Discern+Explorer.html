The DATETIMECMPUTC( ) function returns the number of days between two rdb_date_time columns. The time is ignored.

Syntax

DATETIMECMPUTC(rdb_date_time1, rdb_date_time2, conversion_option)

Metasymbols

rdb_date_time1
A column that stores the higher Julian date value.

rdb_date_time2
A column that stores the lower Julian date value.

Conversion_option

Option

Description

0

No conversion, returns difference in days

1

Adjust the datetime from local to UTC if UTC mode is on, else return as is

2

Adjust the datetime from UTC to local if UTC mode is on, else return as is

3

Adjust the datetime from local to UTC regardless of UTC mode

4

Adjust the datetime from UTC to local regardless of UTC mode

Examples

The following examples assume that the time zone is America / Chicago during daylight savings, making the UTC offset equal to -5 hours:

cmp0 = datetimecmputc(cnvtdatetime(curdate, 0630), cnvtdatetime(curdate-1, 2359),0)

In the above example, cmp0 would be set equal to 0 (zero).

cmp0 = datetimecmputc(cnvtdatetime(curdate, 0630), cnvtdatetime(curdate-1, 1200),0)

In the above example, cmp0 would be set equal to 1 (one).

cmp1 = datetimecmputc(cnvtdatetime(curdate, 0630), cnvtdatetime(curdate-1, 2359),1)

In the above example, cmp1 would be set equal to 0 (zero).

cmp1 = datetimecmputc(cnvtdatetime(curdate, 0630), cnvtdatetime(curdate-1, 1200),1)

In the above example, cmp1 would be set equal to 1 (one).

cmp2 = datetimecmputc(cnvtdatetime(curdate, 0630), cnvtdatetime(curdate-1, 2359),2)

In the above example, cmp2 would be set equal to 1 (one).

cmp2 = datetimecmputc(cnvtdatetime(curdate, 0630), cnvtdatetime(curdate-1, 1200),2)

In the above example, cmp2 would be set equal to 1 (one).

cmp3 = datetimecmputc(cnvtdatetime(curdate, 0630), cnvtdatetime(curdate-1, 2359),3)

In the above example, cmp3 would be set equal to 0 (zero).

cmp3 = datetimecmputc(cnvtdatetime(curdate, 0630), cnvtdatetime(curdate-1, 1200),3)

In the above example, cmp3 would be set equal to 1 (one).

cmp4 = datetimecmputc(cnvtdatetime(curdate, 0630), cnvtdatetime(curdate-1, 2359),4)
