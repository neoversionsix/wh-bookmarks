This will switch the newlines with pipes to make extracting usernames etc easier. Where A2 is the tickets description

=SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(A2, CHAR(10), "|"), CHAR(13), "|"), CHAR(10)&CHAR(13), "|")

To get the first Name:

=MID(D2, SEARCH("|First Name:|", D2) + LEN("|First Name:|"), SEARCH("|", D2, SEARCH("|First Name:|", D2) + LEN("|First Name:|")) - (SEARCH("|First Name:|", D2) + LEN("|First Name:|")))
