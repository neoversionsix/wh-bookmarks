You can use the ASSIGN3( ) function in EKMODULE usage to preassign an expression, then assign the expression to the variable. ASSIGN3( ) first checks the <check_var> to be >= 0 if yes, then it evaluates the <preassign_expr> and lastly moves <from_expr> to <to_expr> and returns <to_expr>.

Syntax

ASSIGN3(<check_var>,<preassign_expr>,<to_expr>,<from_expr>)

Metasymbols

check_var
A user-defined expression.

preassign_expr
An expression.

to_expr
An expression.

from_expr
An expression.

Examples

The following example ...

SET STAT = ASSIGN3(LOGIC_VALID,ASSIGN(CURINDEX,1),LOGIC_RETURN[CURINDEX], SUB1(P1,P2))
