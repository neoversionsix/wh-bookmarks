CNVTB10B16( ) converts a base10 integer to a base16 string with the maximum length defined by length. This function produces values primarily used in bar coding.

Syntax

CNVTB10B16(base10_int,length)

Metasymbols

base10_int

An expression that is a base10 integer.

length

An integer that defines the number of bytes in a string.

Examples

In the following example, X is set to 003039:

X = CNVTB10B16(12345,6)
