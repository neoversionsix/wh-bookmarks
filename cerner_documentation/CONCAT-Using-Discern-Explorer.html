The CONCAT ( ) function appends a string to the beginning or end of an expression, such as a column. For example, to embed a column in quotation marks, concatenate a quotation mark to the beginning and end of the column without truncating trailing spaces.

Syntax

CONCAT(expression,expression,{expression})

Metasymbols

expression

A value of character data type.

Examples

This example shows how to combine two character strings into one. The following statement sets NEWSTRING equal to "CCL IS FUN".

NEWSTRING = CONCAT( "CCL IS " , "FUN")

This example shows how to embed the patient name in quotation marks. Both sets of quotation marks have to be used. A single quotation mark ( ' ) indicates the beginning and end of the concatenate string; double quotation marks ( " ) are the strings being concatenated into the patient name.

NEWNAME = CONCAT(' " ',TRIM(NAME_FULL_FORMATTED),' " ')

This example shows using the Concat() function to combine the first name field from the Person table, a single space, and the last name field from the Person table into a single expression named "Full_Name". The TRIM() function is used to remove trailing space from the first and last names.

FULL_NAME = CONCAT( TRIM(P.NAME_FIRST) , " " , TRIM(P.NAME_LAST))
