DAY( ) returns the day of the month. An integer from 1 to 31 is returned.

Syntax

DAY(julian_date)

DAY(rdb_date_time)

Metasymbols

julian_date

An expression that is anÂ Internal System Date.

rdb_date_time

A column that stores a datetime or function that returns a datetime.

Examples

In this example, if the current date is September 22, 1998, D1 is set to 22. If the update date and time is November 8, 1998, D2 is set to 8.

D1 = DAY(CURDATE)
D2 = DAY(P.UPDT_DT_TM)

In this example, if the active_status_dt_tm is on 15-Aug-2023, active_day is set to 15. If the current date is 15-Aug-2023, cur_day and fun_day are set to 15. 15_day is set to 15.

select
	date = P.ACTIVE_STATUS_DT_TM,
	active_day = day(P.ACTIVE_STATUS_DT_TM),
	cur_day = day(curdate),
	fun_day = day(cnvtdatetime(curdate,0)),
	15_day = day(cnvtdate(08152023))
from
	person p
with maxrec = 10
