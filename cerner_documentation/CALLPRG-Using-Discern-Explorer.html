CALLPRG( ) executes a Discern Explorer (CCL) program and returns the RETURN() statement from the called program. The DECLARE command can be used to define the return data type. By default, the return data type is integer.

Syntax

CALLPRG(program_name[,parameter])

Metasymbols

Program_name
The name of the program object to call. Program_name is passed as a name, string, or variable enclosed in the VALUE() function.

Parameter
A value that is used as parameters in the called program.

Examples

In the below example, TOTAL is set to 22.

DROP PROGRAM TEST:DBA GO
CREATE PROGRAM TEST:DBA

SET TOTAL = CALLPRG(TEST2,2) + 10
CALL ECHO(TOTAL)
END GO
DROP PROGRAM TEST2:DBA GO
CREATE PROGRAM TEST2:DBA
RETURN( 10 + $1)
END GO

The below example uses the DECLARE command to set the return data type to VC, and passes a program_name, string, and variable enclosed in the VALUE() function to the CALLPRG() function.

drop program ccl_test_callprg go
create program ccl_test_callprg

declare ccl_test_callprg2() = vc  with protect ;declare return type for ccl_test_callprg2 as vc
declare username = vc with protect
set username = CALLPRG(ccl_test_callprg2)
CALL ECHO(username)

declare username2 = vc with protect
set username2 = CALLPRG("ccl_TEST_CALLPRG2")
CALL ECHO(username2)

declare prog_name = vc with noconstant("ccl_test_callprg2"), protect
declare username3 = vc with protect
set username3 = CALLPRG(value(prog_name))
CALL ECHO(username3)
end
go
