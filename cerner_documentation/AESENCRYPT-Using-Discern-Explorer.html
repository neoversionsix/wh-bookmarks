The AESENCRYPT ( ) function returns an encrypted text string using Advanced Encryption Standard (AES). The function AESDECRYPT() can be used to decrypt a value that was created using AESENCRYPT().

Syntax

AESENCRYPT(aeskeyid,inbuf,inbuflen[,mode])
AESENCRYPT(customkey,inbuf,inbuflen[,mode])

Metasymbols

aeskeyid
Integer value in the range of 1 to 5.

customkey
A 16-character string used for the encryption key.

inbuf
Character string to encrypt.

inbuflen
Integer value - the length of the character string to be encrypted.

mode
Integer value - limited to 0 (for base-10) or 1 (for base-16).


Examples

In the following example, the encrypt_var is set equal to an encrypted string:

SET ENCRYPT_VAR = AESENCRYPT(1,"ABC12345xyz",TEXTLEN("ABC12345XYZ"),0)


In the example below, the encrypt_var is set equal to five different encrypted strings. Each iteration of the loop causes a different ikey value to be used, resulting in a different encrypted string being returned.

FOR(ikey = 1 to 5)
	SET ENCRYPT_VAR = AESENCRYPT(ikey,"ABC12345xyz",TEXTLEN("ABC12345XYZ"),0)
ENDFOR


In the following example, the encrypt_var is set equal to an encrypted string:

SET ENCRYPT_VAR =
	AESENCRYPT("My16CharacterKey" ,"ABC12345xyz",TEXTLEN("ABC12345XYZ"),0)
