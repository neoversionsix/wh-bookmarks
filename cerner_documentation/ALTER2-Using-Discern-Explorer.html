The ALTER2( ) function determines if the record structure list item is a contiguous record structure segment or a varlist record structure segment then calls the ALTER() or ALTERLIST() function accordingly. Returns 1 if successful, andÂ 0 if not successful. The ALTER2() function was introduced in version 8.6.3 of Discern Explorer. The option to use CALL ALTER2( ) was introduced in version 8.9.2 of Discern Explorer.

Syntax

ALTER2(record_structure_list_item, dimension)
CALL ALTER2(record_structure_list_item, dimension)  ;return type is discarded when using CALL

Metasymbols

record_structure_list_item

A record structure list_item that was defined using the RECORD command.

dimension

An integer representing the total number of positions to define in the record structure list_item.

Examples

In this example, a contiguous record structure segment is created in REC1 with a dimension of 10 and increased in blocks of 10 using the ALTER2() function. A varlist record structure segment is created in REC2. The ALTER2() function also is used to initialize the dimension of this list to 10 and allocate additional blocks of 10 as needed.

DROP PROGRAM TEST1 GO
CREATE PROGRAM TEST1

RECORD REC1 (
	1 QUAL[10] /* creates a contiguous record structure segment */
		2 NAME = C40)

RECORD REC2 (
	1 QUAL[*] /* creates a varlist record structure segment */
		2 NAME = C40)

SELECT INTO "NL:"
	P.NAME_LAST_KEY

FROM	PERSON P

WHERE P.UPDT_DT_TM > CNVTLOOKBEHIND("1,W")

HEAD REPORT
	COUNT = 0
	/* allocate 10 positions in the rec2->qual list*/
	STAT = ALTER2(REC2->QUAL,10)

DETAIL
	COUNT = COUNT + 1
	/*if all available positions of the QUAL lists have been used, add 10 more */
	IF (MOD(COUNT,10) = 1 AND COUNT > 1 )
		STAT = ALTER2(REC1->QUAL,COUNT + 9)
		STAT = ALTER2(REC2->QUAL,COUNT + 9)
	ENDIF
	REC1->QUAL[COUNT]->NAME = P.NAME_LAST_KEY
	REC2->QUAL[COUNT]->NAME = P.NAME_LAST_KEY

FOOT REPORT
	/*resize the QUAL lists down to the number of positions that were used */
	STAT = ALTER2(REC1->QUAL,COUNT)
	STAT = ALTER2(REC2->QUAL,COUNT)

WITH COUNTER
CALL ECHORECORD(REC1)
CALL ECHORECORD(REC2)
END
GO
