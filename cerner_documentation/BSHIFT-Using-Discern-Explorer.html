Introduced in version 8.11.5 of Discern Explorer,Â BSHIFT() performs a bit shift operation on an integer (I4, W8) expression.

Syntax

BSHIFT(integer, shift_cnt)

Metasymbols

Integer
An integer (I4, W8) value.

shift_cnt
If shift_cnt is positive then the integer is shifted to the left. If shift_cnt is negative then the integer is shifted to the right.

Examples

  call echo(bshift(32,1))   ;returns 64  
  call echo(bshift(32,2))   ;returns 128 
  call echo(bshift(32,-1))  ;returns 16  
  call echo(bshift(32,-2))  ;returns 8
