CNVTDATETIMEUTC3() converts a string that represents a date and a time to and from a Universal Coordinated Time (UTC) datetime value. A space is required between the date and time components of the string. Fractional seconds are ignored.

CNVTDATETIMEUTC3() was released with version 8.9.2 of Discern Explorer. Before Discern Explorer versions 8.13.3 and 9.01.3, an information message is returned if the length of the datetime_string is less than the length of the datetime_format and SET TRACE Warning2 is enabled. 

Syntax

CNVTDATETIMEUTC3(datetime_string, datetime_format, [option [,timezone_index]])

Metasymbols

datetime_string
A character string that represents a date and time. A space is required between the date and time components of the string.

Before Discern Explorer versions 8.16.0|9.04.0 the first three characters of the month abbreviation or full month name must match one of the values in the CURLOCALE("MONTHABBREV") string. CNVTDATETIMEUTC3() performs a case and accent insensitive comparison of the first three characters of the month abbreviation or full month name to the CURLOCALE("MONTHABBREV") string to determine the month. If a match is not found, the month with the closest match is returned. Beginning in Discern Explorer versions 8.16.0|9.04.0, if the first three characters of the month abbreviation or full month name do not match one of the values in the CURLOCALE("MONTHABBREV") string, the first four characters of the month abbreviation or full month name are compared to the values in the CURLOCALE("MONTHABBREV2") string. If a match is not found, the full month name is compared to the values in the CURLOCALE("MONTFULL") string. If a match is still not found, the month with the closest match is returned.

datetime_format
A template that describes the format of the datetime_string.

option

Option

Description

0

Noop so just return datetime with no conversion.

1

Convert the datetime from local to UTC if UTC mode is on. Otherwise, return as is.

2

Convert the datetime from UTC to local if UTC mode is on. Otherwise, return as is.

3

Convert the datetime from local to UTC regardless of UTC mode.

4

Convert the datetime from UTC to local regardless of UTC mode.

timezone_index
The index number of the time zone to use for the conversion. Defaults to local time zone. This option allows emulating another target time zone for datetime conversions.

Examples

If the CurTimeZoneApp is America/Chicago, the Call Echo() command in the following example returns 11-Mar-2007 05:30:00 Central Daylight Time if UTC is on:

set my_dt_tm = cnvtdatetimeutc3("20070311 023000","YYYYMMDD HHMMSS",1,35)
call echo(datetimezoneformat(my_dt_tm,curtimezoneapp,"DD-MMM-YYYY HH:mm:ss ZZZZZZ"))

If the CurTimeZoneApp is America/Chicago, and UTC is on, the Call Echo() command in the following example returns the following based on each iteration of the for loop, changing the value of the option parameter used by CNVTDATETIMEUTC3():

10-Mar-2007 20:30:00 Central Standard Time
11-Mar-2007 05:30:00 Central Daylight Time
10-Mar-2007 12:30:00 Central Standard Time
11-Mar-2007 05:30:00 Central Daylight Time
10-Mar-2007 12:30:00 Central Standard Time

for (opt = 1 to 5)
	set option = opt -1
	set my_dt_tm = cnvtdatetimeutc3("20070311 023000","YYYYMMDD HHMMSS",option,35)
	call echo(datetimezoneformat(my_dt_tm,curtimezoneapp,"DD-MMM-YYYY HH:mm:ss ZZZZZZ"))
endfor
